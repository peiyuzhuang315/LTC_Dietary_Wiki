# 口腔乾燥症（Xerostomia / Dry Mouth in Elderly）

> **分類**：口腔健康 ｜ 照護需求 ｜ 進食障礙  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：家屬、照服員、醫護人員、長照業者

---

## 什麼是口腔乾燥症？

**口腔乾燥症**（英文：Xerostomia，俗稱「乾口症」、「口乾症」）是指個人感覺**口腔內持續乾燥**，和/或唾液分泌量明顯減少的症狀或疾病。

口腔乾燥在長照族群中是一個**嚴重低估的問題**——它不只是「口渴」的感覺，而是會直接影響進食能力、吞嚥功能、說話清晰度、牙齒健康和住民整體生活品質。

---

## 台灣老年族群的口腔乾燥現況

**[✅ Fact-Checked]** 老年族群口腔乾燥相關研究：
- 老年人口腔乾燥症狀盛行率：研究估計約 **20-30%**（不同定義下的盛行率差異較大）
- **長照機構住民**的口腔乾燥比例更高（多重慢性病、多重用藥）
- **美國研究**：住院老年患者中，口腔乾燥症是第三大最常見的口腔問題（僅次於蛀牙和牙周病）

## 口腔乾燥的成因

### 1. 藥物副作用（最常見原因）

超過 400 種常見藥物有減少唾液分泌的副作用：

| 常見藥物類型 | 舉例 |
|---|---|
| **抗憂鬱藥** | SSRIs、三環類 |
| **抗組織胺** | 感冒藥、過敏藥 |
| **利尿劑** | 心臟衰竭、高血壓治療 |
| **抗高血壓藥** | Ca channel blockers、ACE inhibitors |
| **抗精神病藥** | 失智症行為症狀（BPSD）治療 |

長照住民普遍多重用藥（Polypharmacy：超過 5 種同時使用），很多住民的口腔乾燥是「多種藥物副作用疊加」的結果。

### 2. 糖尿病

高血糖狀態下，身體為了排出多餘糖分而增加排尿，間接造成脫水和口腔乾燥。

### 3. 頭頸部放療（口腔癌/舌癌/喉癌）

放療對唾液腺的破壞可能是永久性的，這部分患者的口腔乾燥往往最嚴重。

### 4. 自然老化

唾液腺本身會隨老化而功能退化（分泌量下降約 30-40%），即使沒有上述原因也可能自然發生。

## 口腔乾燥對進食的影響

| 影響 | 說明 |
|---|---|
| **食物難以形成食團** | 唾液是食物和唾液澱粉酶混合的媒介，缺乏唾液讓食物難以聚合成可以吞嚥的食團 |
| **食物難以吞嚥** | 乾燥的食物（特別是固態食物）需要更多唾液潤滑才能順暢通過咽喉 |
| **口腔黏膜受傷** | 乾燥讓黏膜脆弱，粗糙食物可能造成黏膜破損 |
| **蛀牙風險增加** | 唾液的抗菌功能降低，口腔細菌大量繁殖，加速齲齒 |
| **食慾下降** | 進食的不舒適感讓住民不想吃東西 |

## 口腔乾燥的照護和飲食對策

### 飲食調整

1. **增加食物的濕潤度**：在食物中加入醬汁、肉汁、湯水；選擇含水量高的食物（豆腐、蒸魚、稀飯）
2. **避免乾性食物**：乾硬的餅乾、麵包、米粉（乾）等應避免或搭配足夠液體進食
3. **少量多次喝水**：提醒住民在用餐中間小口喝水，幫助食物通過

### 口腔護理

1. **人工唾液噴劑**（藥局可購買）：提供短暫的潤濕效果
2. **無酒精型漱口水**（每天使用）：酒精型漱口水會加重乾燥
3. **每餐後刷牙**：口腔乾燥的住民蛀牙風險高，飯後清潔更重要

---

## 對業者的服務意涵

業者設計長照菜單時，應主動考量口腔乾燥住民的需求：
1. 每道菜提供「**加湯版本**」或「**醬汁版本**」，讓廚師可以依住民需求調整
2. 推薦含水量高的食材（豆腐、嫩蛋、燉肉）作為口腔乾燥住民的優先選擇
