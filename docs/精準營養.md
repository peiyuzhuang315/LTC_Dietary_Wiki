# 精準營養（Precision Nutrition）

> **分類**：產品/研發 ｜ 個人化照護 ｜ 科技應用  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：長照業者、醫護人員、家屬

---

## 什麼是精準營養？

**精準營養**（英文：Precision Nutrition 或 Personalized Nutrition）是相對於「一般化飲食建議」提出的新範式——它主張：即使面對同樣的疾病（如糖尿病、肌少症），不同人對相同飲食介入的反應會有顯著差異，因此**最有效的飲食建議應該依據個人的生理特質、基因、腸道菌相、疾病狀態、生活習慣量身定制**，而非套用「所有糖尿病患者都應該這樣吃」的單一標準。

在長照膳食的語境中，精準營養的實踐不一定需要基因檢測（成本太高、多數機構住民不適合），而是**在現有可取得的個人資料（體重、血液檢查、疾病史、藥物使用、吞嚥評估）基礎上，為每位住民設計個人化的飲食計畫**。

## 精準營養的四個資料維度

### 1. 生化指標（Biochemical Markers）

血液檢查數據是精準營養的基礎資料，長照常用指標：

| 指標 | 正常參考範圍（一般成人）| 對膳食設計的意義 |
|---|---|---|
| **白蛋白（Albumin）** | 3.5-5.0 g/dL | < 3.5 提示蛋白質攝取不足/消耗過多 |
| **前白蛋白（Prealbumin）** | 18-40 mg/dL | 更靈敏的近期營養狀態指標（半衰期僅2天）|
| **血紅素（Hemoglobin）** | 男 13-18 g/dL，女 12-16 g/dL | 低於正常值提示貧血，可能需補充鐵、B12、葉酸 |
| **25(OH)D（維生素 D）** | 20-50 ng/mL | 台灣長者普遍低於 20，需補充 |
| **肌酸酐（Creatinine）** | 0.6-1.2 mg/dL | 慢性腎臟病者需限制蛋白質攝取量 |
| **血鉀（Potassium）** | 3.5-5.0 mEq/L | 腎臟病者需限鉀食物（香蕉、番茄等）|
| **血磷** | 2.5-4.5 mg/dL | 洗腎患者需限磷（奶製品、堅果、加工食品）|
| **HbA1c（糖化血色素）** | < 7%（糖尿病患）| 血糖控制狀態，決定碳水化合物攝取策略 |

### 2. 身體組成評估

- **BMI**：體重（kg）÷ 身高²（m²）
  - 老年人適合的BMI：22-26（比一般成人略高，因過瘦對老年人危害更大）
- **BIA（生物電阻抗分析）**：可測量體脂率、肌肉量、骨骼肌指數，是肌少症評估的重要工具
- **小腿圍**：男 < 34 cm / 女 < 33 cm 提示肌少症風險

### 3. 藥物-營養素交互作用

許多長照住民服用多種藥物，某些藥物會直接影響營養素吸收或代謝：

| 藥物 | 影響的營養素 | 因應策略 |
|---|---|---|
| **Warfarin（抗凝血藥）** | 維生素 K | 需固定（不突然增減）深綠色蔬菜攝取量 |
| **ACE 抑制劑（降壓藥）** | 鉀 | 可能造成高鉀，需監控 |
| **利尿劑** | 鉀、鎂、鋅 | 可能造成電解質流失 |
| **PPI（胃藥）** | 鈣、鐵、維生素 B12 | 長期使用影響礦物質和維生素吸收 |
| **Metformin（糖尿病藥）** | 維生素 B12 | 長期使用可能導致 B12 缺乏 |

### 4. 疾病狀態和吞嚥功能

- **吞嚥能力（IDDSI Level）**：決定食物質地
- **腎臟功能（GFR）**：決定蛋白質和礦物質攝取上限
- **糖尿病（HbA1c）**：決定碳水化合物攝取策略
- **心臟衰竭（NYHA 分級）**：可能需要限制液體攝取

## 精準營養在台灣長照的實踐現況

**現實挑戰**：台灣長照機構的現行模式是「大鍋炒」——所有住民吃大致相同的菜色，頂多依失能分級給不同質地的食物，但**個人化的營養計畫（依每位住民的疾病、藥物、血液數據設計的菜單）在多數中小型機構中幾乎不存在**，主要原因是：
1. 沒有足夠的 RD（營養師）人力
2. 個人化設計的操作成本過高
3. 廚房難以在有限人力下執行多種個別菜單

**突破方向**：
- 導入「分組化精準營養」——以 4-6 種標準化飲食計畫覆蓋大多數住民需求，而非真正一人一菜單
- 利用 AI 工具協助菜單設計（目前在研究階段，詳見[AI智慧營養管理](AI智慧營養管理.md)詞條）
- 將精準評估（血液數據 + IDDSI）作為業者 MDT 服務差異化的核心

## 對業者的應用機會

**精準營養是業者服務升級的方向**，而非遙不可及的科技概念：

| 精準化程度 | 做法 | 實施難度 |
|---|---|---|
| **基礎**（今天的標準）| 依 IDDSI 等級提供不同質地食物 | ✅ 可立即執行 |
| **中階**（短期可達）| 依疾病分類（糖尿病/腎臟病/一般）設計 3-4 種菜單變體 | 中等 |
| **進階**（中長期目標）| 依個人血液數據+藥物+IDDSI整合設計月度飲食計畫（需RD常駐或密切合作）| 較高 |
