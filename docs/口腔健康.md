# 詞條：口腔健康（Oral Health in Aging）
> **所屬面向**：使用者/照護需求面向 | **改寫日期**：2026-03-18

---

「牙壞了補一補就好」——這句話換到長照場域會引發一連串的多米諾骨牌效應。口腔健康在老年人身上不只是牙齒的問題，而是整個「吃飯能力」的上游開關：**牙齒少了 → 咀嚼效率降低 → 食物選擇縮窄 → 蛋白質攝取不足 → 肌少症加速 → 吞嚥肌群退化 → 吸入性肺炎風險升高**。這條鏈，任何一個環節都跟業者的膳食設計高度相關。

### 台灣長照住民的口腔健康現況

**[✅ Fact-Checked]** 衛福部 2021-2023 年調查：台灣 65 歲以上長者中 **36.6%** 自然牙少於 20 顆。
> 來源：[mohw.gov.tw](https://mohw.gov.tw)

台灣的長期照護評鑑制度近年加強了口腔健康的稽查，要求機構記錄每位住民的口腔衛生管理情況，包括每日口腔清潔頻率、義齒管理記錄等。這些要求對業者的意義是：**你的食物設計如果能同步考慮住民的口腔狀況，會直接幫助機構通過評鑑的口腔管理項目**。

### 不同牙齒狀況需要的食物調整

| 口腔狀況 | 食物設計建議 | 對應 IDDSI 等級（參考）|
|---|---|---|
| **牙齒完整、口腔功能良好** | 正常食物即可，注重蛋白質密度 | IDDSI 7 |
| **缺牙 5-10 顆（未裝義齒）** | 食物切小塊、避免韌性高的肉 | IDDSI 6-7 |
| **裝全口假牙但適應性差** | 軟質食物，避免黏性高（如麻糬、年糕）| IDDSI 5-6 |
| **牙周病嚴重、咬合力下降** | 細碎濕軟，食材刀工精細 | IDDSI 5 |
| **口乾（Xerostomia）** | 增加醬汁含水量、避免乾燥食物 | 質地設計搭配液體補充 |

### 業者可主動提供的口腔健康支持服務

口腔健康衛教不是業者的傳統業務範圍，但主動提供以下材料，可以建立超出期待的服務深度：

1. **「咀嚼能力 × 食物選擇」衛教單張**（A4 單頁，護理人員可向家屬說明）
2. **月度膳食設計說明**：當月菜單的「口腔友善」說明（哪些菜特別適合義齒住民、哪些需謹慎）
3. **健口操圖卡**（口腔運動，改善殘存口腔肌功能）—— 可與 SLP 合作設計
