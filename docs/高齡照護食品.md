# 詞條：高齡照護食品（Geriatric Care Food）
> **所屬面向**：使用者/照護需求面向 | **改寫日期**：2026-03-18

---

「高齡照護食品」是比「介護食」更廣義的概念——它不只是針對吞嚥障礙者的質地調整食品，而是**所有為了支持高齡者健康維持、疾病管理、生活質量提升而特別設計的食品**的總稱。從預防性的高蛋白優格，到重度失能者的均質流質，都在這個範疇裡。

**[高齡照護食品](高齡照護食品.md)** 在台灣法規上沒有對應的特定品類名稱（食安法把它涵蓋在「特殊膳食食品」或「一般食品」下，依功效聲稱而定），但在市場層面，這個詞已被學術界、照護媒體和業者廣泛使用。

### 高齡照護食品的五個設計維度

無論是機構供膳還是居家補充品，優秀的高齡照護食品設計必須同時在五個維度達標：

| 設計維度 | 核心問題 | 達標工具 |
|---|---|---|
| **安全性** | 住民能否安全吞嚥？ | IDDSI 質地等級 + SLP 評估 |
| **營養密度** | 每口食物含多少有效營養？ | 蛋白質 ≥ 1g/kg/天，熱量密度 ≥ 1kcal/g |
| **適口性** | 住民願不願意吃？ | 天然鮮味、保留香氣、嫩化技術 |
| **合規性** | 符合哪些認證標準？ | Eatender、GHP/HACCP、食安法標示 |
| **可及性** | 院長/家屬能否負擔且穩定取得？ | 定價策略、通路布局 |

**[✅ Fact-Checked]** 台灣衛福部「特定疾病配方食品（Non-oral-use）」和「口服鼻胃管灌食配方食品」屬於高齡照護食品中最嚴格管制的子類別，須取得衛福部核可，標示和廣告有嚴格規範。
> 來源：[fda.gov.tw食藥署](https://fda.gov.tw)

### 業者定位的核心選擇

高齡照護食品業者往往需要在「**市場廣度**」和「**品類深度**」之間做選擇：

- **廣度型**（如護力養）：一個全方位均衡配方，通吃機構 + 居家 + 藥局，靠服務深度制勝
- **深度型**（如漢典食品）：150+ 道配方，針對不同機構、不同疾病、不同質地需求分別開發
- **技術型**（如格林全食物）：聚焦特定技術（TFL），在此技術下建立深度產品壁壘
