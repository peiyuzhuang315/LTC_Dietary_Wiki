# 詞條：漢典食品（Han-Dian Food Co.）
> **所屬面向**：業者詞條 | **改寫日期**：2026-03-18

---

如果你想知道台灣長照冷凍調理食品業者的「天花板」長什麼樣，大部分業內人士的答案是：**漢典食品**。這家公司從彰化田中的冷凍食品工廠出發，花了20年時間把銀髮友善食品做成一門技術壁壘極高、醫院願意掛名技術合作的產業。

**[✅ Fact-Checked]** 漢典食品（總公司：彰化縣田中鎮）基本資料：
- **總經理**：鍾紀銘
- **技術特長**：以低溫真空氣泡脫氣技術降低食品均質化後的空氣含量（減少嗆咳風險），搭配急速冷凍維持解凍後質地穩定
- **認證**：**ISO 22000 + HACCP 雙認證**
- **產品規模**：超過 **1,000 道配方**、150 種以上銀髮友善調理包
- **技術合作夥伴**：臺大醫院雲林分院（復健科+營養科）、承業生醫集團
- **海外布局**：英國設廠（已獲歐洲市場出口認可）
> 來源：[frozenfoods.com.tw](https://frozenfoods.com.tw)

### 三個讓漢典成為標竿的關鍵決策

| 決策 | 時間點（估計）| 商業意義 |
|---|---|---|
| 主動尋求臺大醫院技術合作 | 2010 年代中期 | 取得「醫院推薦」背書，打開院線採購市場 |
| 投入 ISO 22000 + HACCP 雙認證 | 多年前 | 建立難以複製的採購門檻，排除中小競爭者 |
| 英國設廠、打進歐洲市場 | 2020 年代 | 建立「台灣長照食品可以出口」的市場示範效應 |

### 業者從漢典學什麼？

漢典的成功沒有捷徑，但有幾個核心學習點：**①** 從一開始就把「臨床合作」放在商業策略最前位——不是先有產品再去找醫院背書，而是和醫院一起開發；**②** 雙認證（ISO 22000 + HACCP）是護城河而不是負擔，它把大量沒有資源做認證的競爭者排除在最高價值客群之外；**③** 出口市場從一開始就是戰略佈局，而不是等台灣市場飽和後才想到。
