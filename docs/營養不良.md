# 營養不良 (Malnutrition)

> **分類**：照護需求｜臨床症狀｜膳食設計基礎  
> **更新日期**：2026-03-31 ｜ **適合閱讀對象**：家屬、照服員、長照業者、醫護人員

---

## 什麼是營養不良？

營養不良（Malnutrition）是指身體攝取、吸收或利用營養素的能力失衡，導致能量、蛋白質或其他必需營養素的不足或過剩，進而對身體組織、器官功能及臨床結果產生負面影響的一種狀態。在長照領域，我們主要關注的是「營養不足」，特別是蛋白質熱量營養不良（Protein-Energy Malnutrition, PEM）及微量營養素缺乏。

營養不良並非單純的「吃不夠」，它是一個複雜的臨床問題，可能由多種因素引起，包括疾病、藥物副作用、吞嚥困難、牙口問題、消化吸收功能下降、心理社會因素（如憂鬱、失智、獨居）以及不適當的飲食供應等。

對長照現場而言，營養不良的直接影響是 **加速身體功能退化、增加感染風險、延緩傷口癒合、提高跌倒機率、延長住院時間，並顯著降低長輩的生活品質**。

營養不良的嚴重程度可依據多種評估工具進行分類，例如 Mini Nutritional Assessment (MNA) 或 Malnutrition Universal Screening Tool (MUST)。以 MNA 評估為例：

| 分類             | MNA 總分 (滿分 30) | 臨床意義                                     |
| :--------------- | :----------------- | :------------------------------------------- |
| 營養良好         | 24-30 分           | 無營養不良風險                               |
| 營養不良風險     | 17-23.5 分         | 存在營養不良風險，需密切監測及介入           |
| 營養不良         | < 17 分            | 已有營養不良，需立即且積極的營養介入與治療 |

## 為什麼重要

營養不良是長照服務中一個被低估卻極為關鍵的問題，其影響層面廣泛，對長輩、家屬、照服員及長照機構都帶來沉重負擔。

> 營養不良不僅是生理問題，更是影響長輩生活品質、照護成本與家庭負擔的全面性挑戰。早期發現與積極介入，是提升長照品質的基石。

*   **對長輩的影響**：營養不良會導致肌肉流失（肌少症）、骨質密度下降、免疫力降低、傷口癒合緩慢、認知功能退化，進而增加跌倒、感染、壓瘡及再住院的風險。這些都嚴重影響長輩的獨立生活能力與尊嚴，使其更容易陷入失能與臥床的惡性循環，最終大幅降低生活品質。

*   **對家屬的影響**：當長輩因營養不良導致健康惡化，家屬需投入更多時間與心力照護，承受更大的心理壓力與經濟負擔。反覆的住院與出院，不僅打亂家庭生活步調，更可能導致家屬因長期照護壓力而身心俱疲，甚至影響其工作與社交生活。面對長輩健康持續惡化，家屬也常感到無助、焦慮與自責，加劇家庭內部衝突。此外，額外的醫療費用、營養補充品開銷，以及可能衍生的看護費用，都對家庭經濟造成沉重壓力。

*   **對照服員與長照機構的影響**：營養不良的長輩通常需要更複雜、更密集的照護。這意味著照服員的工作量增加，例如協助進食、處理壓瘡、管理感染症狀、頻繁更換尿布等。對於長照機構而言，營養不良問題若未能有效管理，可能導致住民健康狀況不佳、滿意度下降，進而影響機構的聲譽與入住率。同時，處理相關併發症所需的醫療資源與人力成本也會大幅提升，甚至可能面臨主管機關的查核與評鑑壓力。積極的營養管理不僅能提升住民的健康，也能優化照服員的工作效率，並彰顯機構的專業照護品質。

*   **對醫療系統與社會的影響**：長輩因營養不良導致的併發症，如肺炎、泌尿道感染、骨折等，往往需要反覆住院治療，這不僅佔用寶貴的醫療資源，增加健保支出，也延長了病床周轉率，影響醫療系統的運作效率。從社會層面來看，一個普遍存在營養不良問題的長者群體，將會增加社會的整體照護負擔，降低高齡人口的健康壽命與生活品質，對國家長照政策的推動構成巨大挑戰。因此，將營養不良視為重要的公共衛生議題，並積極推動預防與介入策略，是實現健康老齡化的關鍵。
