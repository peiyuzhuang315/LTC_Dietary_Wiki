# 詞條：營養師（Registered Dietitian / RD）
> **所屬面向**：團隊/專業面向 | **改寫日期**：2026-03-18

---

在 MDT 團隊的分工裡，SLP 決定「住民能安全吃什麼質地」，而**[[營養師]]**（Registered Dietitian，RD）決定的是「住民的每一口食物要把哪些營養送進身體」。這兩個角色缺一不可——吞嚥安全和營養充足，是長照膳食服務的兩張入場券。

### 營養師在長照膳食中的七大核心工作

| 工作項目 | 具體內容 | 頻率 |
|---|---|---|
| **個案營養評估** | 測量體重、BMI、小腿圍，執行 MNA-SF，評估基礎代謝率 | 入住時 + 每季 |
| **個人化飲食處方** | 計算熱量需求（Harris-Benedict 公式）、蛋白質目標、禁忌食材 | 入住時 + 每季更新 |
| **菜單設計審查** | 審核業者提供的菜單，確認三大營養素比例符合 DRIs 建議 | 每季 |
| **特殊飲食規劃** | 糖尿病、腎臟病、吞嚥障礙等特殊需求的飲食調整 | 隨需求 |
| **進食量追蹤** | 監控住民的每日進食量，發現食量下滑立即介入 | 每月 |
| **補充品選擇建議** | 依住民個別狀況推薦適合的口服營養補充品（如護力養）| 每季評估 |
| **家屬衛教** | 說明住民的飲食計畫，建立家屬對業者食物設計的信任 | 每次家庭會議 |

**[✅ Fact-Checked]** 台灣「老人福利機構設施設備標準」規定：設有廚房的養護機構（50 床以上）應聘有或委任兼任營養師。
> 來源：衛福部社家署相關法規

**[✅ Fact-Checked]** 台灣知名長照營養師代表：**陳珮蓉**（台灣長期照護服務領域知名 RD），專注老年營養及 IDDSI 應用推廣。
> 來源：業界資訊

### 業者如何利用 RD 建立服務深度？

- **委任兼任 RD**（月費顧問制，約每月 5,000-15,000 元），負責為業者服務的機構住民出具季度個案營養報告
- **聯名設計菜單**：RD 在菜單上掛名「本菜單由 RD XXX 審核」，大幅提升護理長和家屬對業者食物的信任度
