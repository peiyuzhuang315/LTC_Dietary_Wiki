# 口腔衰弱（Oral Frailty）

> **分類**：老年醫學 ｜ 照護需求 ｜ 口腔健康  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：家屬、照服員、長照業者、牙科醫師

---

## 什麼是口腔衰弱？

**口腔衰弱**（英文：Oral Frailty）是一種以**口腔機能逐漸退化**為核心特徵的老年症候群，包含咀嚼能力下降、吞嚥功能退化、口腔乾燥、舌頭力量減弱、溝通表達困難等多個面向的複合性退化。它在 2016 年由日本東京大學**飯島勝矢教授**研究團隊正式提出概念框架，現已成為老年照護領域的重要評估指標。

「口腔」在人體功能裡的重要性超乎多數人的認知——**口腔機能的退化，往往是全身功能退化的前哨信號**。研究發現，口腔衰弱的老年人發展為全身衰弱、需要長照的風險，是口腔機能正常者的 **2.4 倍**；死亡風險則高達 **2.09 倍**。

---

## 口腔衰弱 vs 口腔健康問題——這兩者不一樣

很多人直覺認為「口腔問題就是蛀牙或牙周病」，但口腔衰弱強調的是**口腔「機能」的退化**，而非只是結構性的問題：

| 面向 | 口腔結構問題 | 口腔衰弱（機能退化）|
|---|---|---|
| 主要評估對象 | 牙齒、牙齦、黏膜 | 咀嚼力、舌壓、吞嚥速度、口腔動作協調 |
| 主責醫療人員 | 牙科醫師 | 語言治療師、牙科醫師、老年醫學科 |
| 介入方式 | 補牙、洗牙、假牙 | 口腔運動訓練、IDDSI 飲食調整 |
| 對長照的影響 | 疼痛影響進食 | 安全進食能力下降 |

口腔衰弱的概念把牙科和長照照護**真正串聯起來**——口腔不只是牙齒的問題，而是全身功能的起點。

## 口腔衰弱的七個退化面向

飯島勝矢團隊定義的「口腔衰弱」包含以下幾個可量化的退化指標：

| 評估面向 | 說明 | 正常參考值 |
|---|---|---|
| **殘存牙齒數** | 自然牙齒數量 | ≥ 20 顆（80-20 運動目標）|
| **咀嚼力** | 可咀嚼的食物硬度 | 可咀嚼花生、核桃等堅硬食物 |
| **舌壓** | 舌頭最大推壓力量 | ≥ 30 kPa（舌壓計測量）|
| **嘴唇閉合力** | 嘴唇最大閉合力 | 依標準設備測量 |
| **口腔濕潤度** | 唾液分泌量 | 黏膜潤濕程度評估 |
| **咬合力** | 雙側最大咬合壓力 | 依牙合紙或測量設備評估 |
| **吞嚥功能** | 反覆唾液吞嚥測試（RSST）| 30 秒內吞嚥 ≥ 3 次 |

## 台灣口腔衰弱現況

**[✅ Fact-Checked]** 台灣相關研究：
- 台灣約有超過 **70% 的 65 歲以上長者**有程度不等的口腔機能退化跡象
- 長照機構住民的口腔衰弱問題更為嚴峻：**定期接受口腔照護的機構住民比例仍偏低**
- 台灣八十歲長者平均殘存牙齒數：約 **10-12 顆**（距80-20目標差距仍大）
> 來源：衛福部口腔健康計畫、[ankecare.com](https://ankecare.com)

**「80-20 運動」**：台灣衛福部推廣的目標——讓80歲長者至少保有20顆功能性牙齒。足夠的牙齒數量是維持正常咀嚼功能、預防口腔衰弱的基礎。

## 如何辨識口腔衰弱早期警訊？

### 家屬、照服員可以觀察的 10 個信號

| 警訊 | 可能意義 |
|---|---|
| ① 吃飯時間越來越長 | 咀嚼效率下降 |
| ② 開始挑食（拒絕過硬的肉類、蔬菜）| 咬合力或咀嚼力不足 |
| ③ 說話越來越不清楚（口齒不清）| 舌頭和口唇肌肉協調退化 |
| ④ 常常嗆到 | 吞嚥反射速度變慢 |
| ⑤ 口角常有口水流出 | 嘴唇閉合力下降 |
| ⑥ 覺得嘴巴乾（常要喝水）| 唾液腺功能退化 |
| ⑦ 假牙開始鬆動、不舒服 | 口腔結構改變，假牙不合 |
| ⑧ 食物殘渣常留在口腔內 | 自我口腔清潔能力下降 |
| ⑨ 口臭加重 | 口腔清潔能力下降 + 菌量增加 |
| ⑩ 主動減少社交飲食活動 | 對進食的信心和愉悅感下降 |

## 口腔衰弱的惡性循環

口腔衰弱具有典型的「惡性循環」結構，一旦啟動往往自我強化：

```
咀嚼力下降 → 避免硬質食物 → 攝取多樣性降低
                ↓
         蛋白質和微量元素攝取不足
                ↓
         肌肉（含口腔肌肉）進一步退化
                ↓
         咀嚼力進一步下降 ↑（循環）
```

打破這個循環的關鍵，是同時介入「口腔機能訓練」和「飲食質地調整」——兩者缺一不可。

## 預防與介入策略

### 1. 口腔運動訓練（詳見[口腔運動](口腔運動.md)詞條）
- **舌頭訓練**：伸舌、舌根用力、舌頭水平移動
- **嘴唇訓練**：嘴唇用力緊閉、發「啪啪啪」音
- **臉頰訓練**：鼓氣、吹氣

每天 10-15 分鐘，4-8 週後可見到舌壓、咬合力的改善。

### 2. 牙齒保健
- 每半年至一年的牙科定期檢查
- 假牙定期重新調整確保密合
- 每日口腔清潔（無法刷牙者使用口腔護理海綿棒）

### 3. 飲食介入
- 質地調整（IDDSI Level 5-6 軟質一口量食物）
- 增加蛋白質攝取強化口腔肌肉
- 補充維生素 D（影響口腔肌肉功能和骨骼）

### 4. 唾液腺刺激
- 按摩唾液腺（耳下腺、頜下腺）
- 使用人工唾液噴劑（口腔乾燥嚴重者）
- 減少導致口腔乾燥的藥物（需與醫師討論）

## 對長照膳食業者的商業意義

**口腔衰弱是MDT服務最好的敲門磚**：口腔衰弱評估（Oral Frailty Index）的結果，直接告知業者應該供應**哪個IDDSI等級**的食物，也讓業者有明確的科學依據向院長說明「為什麼我的食物需要這樣的設計」。

業者可協助機構建立的服務閉環：
1. **初始評估**：引進或協助SLP使用口腔衰弱評估工具（Oral Frailty Index-8）對所有住民篩查
2. **飲食設計**：依評估結果分配住民至對應 IDDSI 等級
3. **定期追蹤**：每季重新評估，記錄口腔功能改善或退化趨勢
4. **成效呈現**：以口腔衰弱分數變化作為KPI，向院長回報服務成效
