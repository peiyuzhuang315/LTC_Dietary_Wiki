# 長照給付

> **分類**：長照知識庫｜費用與給付  
> **更新日期**：2026-03-26 ｜ **適合閱讀對象**：家屬、照服員、長照業者、醫護人員

---

## 什麼是長照給付？

長照給付，指的是政府為減輕失能者及其家庭的照顧負擔，透過「長期照顧服務法」及相關政策（如長照2.0），提供符合資格的失能者各項照顧服務的費用補助。這項給付旨在協助失能者在家中或社區獲得所需的照顧，提升生活品質，並支持家庭照顧者，避免因照顧壓力而影響家庭功能。

長照給付並非直接發放現金，而是以「服務給付」為主，由政府依據失能者的需求評估結果，核定可使用的服務項目與額度，再由民眾選擇合約的服務單位提供服務，政府則支付大部分費用，民眾僅需負擔部分費用（即部分負擔）。

## 誰可以申請長照給付？（服務對象）

符合以下任一資格的失能者，經評估後可申請長照給付：

1.  **65歲以上** 的失能老人。
2.  **55歲以上** 的原住民。
3.  **領有身心障礙證明（手冊）** 的失能者。
4.  **僅工具性日常生活活動（IADLs）失能** 且獨居的長者。
5.  **失智症患者**：經醫師診斷為輕度以上失智症者。

**失能程度評估：**
申請者需經地方政府長期照顧管理中心（照管中心）的照管專員到府評估，依據「日常生活活動功能量表（ADLs）」及「工具性日常生活活動功能量表（IADLs）」等指標，核定失能等級（共8級），並據此核定給付額度與服務內容。

## 長照給付項目有哪些？

長照2.0服務項目主要分為四大類，給付額度會依失能等級而異：

1.  **照顧及專業服務（A碼、B碼、C碼服務）**
    *   **居家照顧服務**：身體照顧（如沐浴、穿衣、翻身）、家務協助（如備餐、打掃、代購）。
    *   **日間照顧服務**：在社區據點提供日間照顧、活動參與、餐食、復健等。
    *   **家庭托顧服務**：在照顧服務員家中接受照顧。
    *   **專業服務**：居家護理、居家復健、營養照護、失智症照護、吞嚥訓練等。
    *   **喘息服務**：提供短期替代照顧，讓家庭照顧者獲得休息。

2.  **交通接送服務**
    *   協助失能者往返醫療院所就醫或復健。

3.  **輔具服務及居家無障礙環境改善服務**
    *   **輔具服務**：補助購買或租賃輪椅、助行器、氣墊床等輔具。
    *   **居家無障礙環境改善**：補助居家環境改造，如加裝扶手、改善浴廁、調整門檻等。

4.  **喘息服務**
    *   當家庭照顧者因故無法照顧失能者時，提供短期替代照顧，讓照顧者得以休息。服務方式包含居家喘息、機構喘息、日間照顧中心喘息、巷弄長照站喘息等。

## 長照給付額度與部分負擔

長照給付的額度會依據失能等級（1-8級）而有不同，失能程度越嚴重，每月可使用的給付額度越高。同時，政府會依據申請者的家庭經濟狀況，區分不同的部分負擔比例：

| 服務項目       | 一般戶（自付比率） | 中低收入戶（自付比率） | 低收入戶（自付比率） |
| :------------- | :----------------- | :--------------------- | :----------------- |
| 照顧及專業服務 | 16%                | 5%                     | 0%                 |
| 交通接送服務   | 21%                | 7%                     | 0%                 |
| 輔具及居家無障礙環境改善 | 30%                | 10%                    | 0%                 |
| 喘息服務       | 16%                | 5%                     | 0%                 |

**說明：**
*   **一般戶**：家庭總收入平均每人每月未達當地區消費支出160%者。
*   **中低收入戶**：符合社會救助法之中低收入戶資格者。
*   **低收入戶**：符合社會救助法之低收入戶資格者。
*   實際給付額度會由照管專員評估後，依據核定的照顧計畫而定。超出核定額度的費用，需由民眾全額自費。
