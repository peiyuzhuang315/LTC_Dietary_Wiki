# 增稠劑（Thickening Agents）

> **分類**：產品/研發 ｜ 照護工具 ｜ 吞嚥安全  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：家屬、照服員、護理師、長照業者

---

## 什麼是增稠劑？

**增稠劑**（英文：Thickening Agents）是一種加入液體後，能讓液體變得更黏稠的食品添加物。在長照和醫療照護的脈絡中，增稠劑的核心目的是：**讓吞嚥障礙患者能夠更安全地飲用液體**，降低液體誤入氣管（誤吸）的風險。

**為什麼液體比固體更危險？**

對吞嚥障礙患者而言，稀薄的液體（如清水、湯汁、果汁）反而比固體食物更容易造成誤吸——因為稀薄液體流動速度快，咽喉來不及關閉保護氣管。增稠劑讓液體「放慢腳步」，給吞嚥肌群更多時間完成協調動作。

這也是 IDDSI 框架中，**液體等級（Level 0-4）和固體等級（Level 3-7）分開設計的原因**——同一位住民可能固體可以吃到 Level 6，但液體必須增稠至 Level 3。

---

## 增稠劑的種類

市面上的長照增稠劑主要分為兩大類：

### 1. 澱粉類增稠劑（Starch-Based）

| 特性 | 說明 |
|---|---|
| 來源 | 玉米澱粉、木薯澱粉等 |
| 增稠原理 | 澱粉顆粒吸水膨脹，增加液體黏度 |
| **優點** | 成本低廉 |
| **缺點 1** | **受溫度影響大**：熱液體中增稠效果強，冷卻後繼續變稠（難以控制稠度）|
| **缺點 2** | **受唾液影響**：口腔中的澱粉酶（Amylase）分解澱粉，液體在口腔中會**越來越稀**，增加誤吸風險 |
| 適用情境 | 成本優先考量、短期使用 |

### 2. 膠體類增稠劑（Gum-Based，主流選擇）

| 特性 | 說明 |
|---|---|
| 常見成分 | **黃原膠（Xanthan Gum）**、關華豆膠（Guar Gum）|
| 增稠原理 | 多醣類分子在水中形成網狀凝膠結構，均勻增稠 |
| **優點 1** | **不受溫度影響**：冷熱飲料中稠度表現一致 |
| **優點 2** | **不受唾液影響**：口腔中不會被分解，稠度穩定 |
| **優點 3** | 少量即可達到目標稠度（劑量精確可控）|
| 適用情境 | **長照臨床首選**，特別適合需要精確稠度管理的住民 |
| **注意事項** | 部分研究指出黃原膠**不適合新生兒**（不同機制，成人安全）|

**[✅ Fact-Checked]** 目前台灣市面上主流的長照增稠劑（如百仕可 Thick-It、多樂力 ThickenUp Clear 等）均以**黃原膠為主要成分**，是臨床吞嚥治療師普遍推薦的選擇。

## IDDSI 液體稠度等級說明

IDDSI 將液體分為 5 個稠度等級（Level 0-4），增稠劑的用量決定達到哪個等級：

| IDDSI 等級 | 名稱 | 描述 | 使用場景 |
|---|---|---|---|
| **Level 0** | 稀薄液體（Thin）| 一般清水、果汁 | 無吞嚥障礙者 |
| **Level 1** | 微稠液體（Slightly Thick）| 比清水略稠 | 需要輕微減緩流速 |
| **Level 2** | 低稠液體（Mildly Thick）| 類似牛奶、nectar | 輕度咽喉期障礙 |
| **Level 3** | 中稠液體（Moderately Thick）| 類似蜂蜜，可緩慢從湯匙流下 | 中度吞嚥障礙 |
| **Level 4** | 高稠液體（Extremely Thick）| 類似布丁，不會從湯匙自行流下 | 重度吞嚥障礙 |

## 如何正確使用增稠劑？

### 基本操作原則

1. **先量再加**：不同品牌的增稠劑濃度不同，嚴格按照說明書的建議用量，避免猜測
2. **充分攪拌**：加入增稠劑後需充分攪拌至溶解均勻，避免結塊
3. **等待稠化**：黃原膠增稠劑通常需要 **30-60 秒**完全稠化，攪拌後稍待再測試
4. **保持一致性**：每次使用相同的量、相同的液體量，確保稠度穩定

### 常見錯誤

| 錯誤行為 | 後果 |
|---|---|
| 用「感覺差不多」估計增稠劑用量 | 稠度不穩定，某次可能稠度不足，造成誤吸 |
| 加入熱飲後立即給住民喝 | 膠體增稠劑未充分稠化，初始稠度不達標 |
| 一次沖泡大量存放 | 時間越久稠度越高（過稠影響口感接受度）|
| 把澱粉類增稠劑加入溫熱湯品 | 冷卻後繼續變稠，難以控制稠度等級 |

### 增稠劑對口感的影響

老實說，增稠後的液體**幾乎沒有任何一位住民表示「好喝」**——黏稠的液體口感陌生，剝奪了「喝一口清涼的水」這個生命中最基本的享受之一。這是為什麼「[尊嚴進食](尊嚴進食.md)」和「飲食滿意度」在長照研究中非常強調：即使吞嚥安全有其必要性，照護者也應該用各種方式（口味選擇、溫度調整、合適容器）盡量讓增稠飲品仍有一定的接受度。

## 增稠劑的購買與費用參考

**[⚠️ 待查證]** 台灣市售增稠劑概估費用：
- 澱粉類增稠劑：約 200-350 元/罐（250-400g）
- 黃原膠增稠劑：約 400-800 元/罐（125-200g）

費用看起來比澱粉類高，但黃原膠增稠劑的每份使用量較少，因此每次使用的實際成本差距不如容量差距那麼大。

長照機構大量採購可向業者洽談批量優惠；居家使用者也可透過輔具補助等管道詢問是否有補助項目。

## 對長照業者的產品開發啟示

部分長照膳食業者開始將「液體稠化服務」整合進供膳服務中——即在交付的液體食品（如果汁、牛奶、湯品）中，依機構 SLP 開立的稠度等級，預先增稠至正確等級後出貨。這個做法的優點：
1. 消除照服員現場增稠的操作誤差風險
2. 業者可以明確標示「每份液體已達 IDDSI Level X」，強化食安合規聲稱
