# 詞條：營養宣稱（Nutrition Claims）
> **所屬面向**：產品/研發面向 | **改寫日期**：2026-03-18

---

「高蛋白、低脂、低鈉——這些標示合法嗎？」是業者在設計產品包裝時最常問的問題之一。**[[營養宣稱]]** 是食品在包裝上對特定營養素含量做出說明或聲稱的行為，在台灣受到食安法和「包裝食品宣稱相關法規」的嚴格規範。

### 台灣食品合法營養宣稱的三個類別

| 宣稱類型 | 說明 | 使用門檻 |
|---|---|---|
| **含量宣稱** | 聲稱某成分「高」「低」「無」「減少」等 | 必須符合衛福部規定的數值門檻 |
| **比較宣稱** | 比較「比 XXX 少 25% 脂肪」等 | 需指明比較對象，且差異須真實可驗證 |
| **功效宣稱** | 說明某成分對身體健康的幫助 | 只有健康食品認證產品才能合法聲稱 |

**[✅ Fact-Checked]** 衛福部食藥署公告的「高蛋白」宣稱門檻：
- **「高蛋白」**：每 100 大卡含蛋白質 ≥ **10 克**（固體）；或每 100g 含蛋白質 ≥ **10 克**
- **「低脂」**：每 100g 脂肪含量 ≤ **3 克**（固體）
- **「低鈉」**：每 100g 鈉含量 ≤ **120 毫克**
> 來源：[fda.gov.tw食藥署](https://fda.gov.tw)

### 三個業者常踩的宣稱地雷

1. **「幫助維持骨骼健康」**——非健康食品認證不可用，要改成「鈣質有助於維持骨骼正常發育」（這是食藥署核准的一般性標示說明）
2. **「改善吞嚥，適合長照住民」**——指向特定疾病應用的聲稱，不符合一般食品規範
3. **「適合肌少症患者」**——指向疾病名稱，違反食安法
