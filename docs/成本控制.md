# 詞條：成本控制（Cost Control）
> **所屬面向**：經營/營運面向 | **改寫日期**：2026-03-18

---

長照膳食業者最常被院長殺價的那個瞬間，往往是因為雙方都不清楚「真實成本在哪裡」。院長只看到「一份飯 85 元，感覺很貴」；業者則清楚知道，這 85 元裡面有食材、冷凍包裝、冷鏈運費、HACCP 認證攤提、SLP 顧問費……**[成本控制](成本控制.md)** 不只是讓你少花錢，更是讓你在價格談判時能夠講出一個有邏輯的故事。

### 長照膳食成本的七個構成項

| 成本項目 | 佔比估算 | 可控程度 | 降低策略 |
|---|---|---|---|
| **食材採購** | 35-45% | 高 | 集中採購、建立供應商議價框架 |
| **人力（廚師/品管）** | 20-30% | 中 | 標準化 SOP 減少技術依賴 |
| **冷鏈物流** | 10-20% | 中 | 路線優化、車輛共享 |
| **包裝材料** | 5-10% | 中等 | 規模採購、永續材料替代 |
| **認證/合規攤提** | 2-5% | 低（必須投入）| 視為品牌護城河投資 |
| **設備折舊** | 3-7% | 低 | 正確預估折舊排程 |
| **管銷/業務** | 5-10% | 高 | 聚焦 ROI 最高通路 |

### 冷鏈是最容易低估的坑

**[✅ Fact-Checked]** 冷鏈物流成本比常溫物流高約 **3-4 倍**，冷凍宅配常見費用約 **250-400 元/箱**。對居家送餐業者來說，物流成本佔整體服務成本比例可能高達 **40%**——這就是為什麼[銀色大門](銀色大門.md)以「路徑最佳化 SaaS」為核心競爭力，每降低 10% 送餐時間就能在不增加人力的情況下服務更多住民。
> 來源：[ankecare.com](https://ankecare.com)

### 自煮 vs 外包：50 床機構的真實數字

這是業者說服院長的最核心計算，要準備現成的試算框架：

| 成本項目 | 機構自煮（50 床）| 業者外包（50 床）|
|---|---|---|
| 廚師人力（2 名，含勞健保）| 月 70,000 元 / 年 840,000 元 | 0 |
| 食材採購（無集中議價）| 年估約 240,000 元 | **含在外包費中** |
| 設備折舊 | 年約 30,000 元 | 0 |
| 水電 | 年約 24,000 元 | 0 |
| 外包費 | — | **年約 420,000-510,000 元** |
| **合計** | **年約 940,000 元+** | **年約 420,000-510,000 元** |
| **潛在節省** | — | **年約 430,000-520,000 元** |
