# 中央廚房（Central Kitchen）

> **分類**：經營/營運 ｜ 生產管理 ｜ 食安管理  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：長照業者、機構管理者、投資者

---

## 什麼是中央廚房？

**中央廚房**（英文：Central Kitchen 或 Commissary Kitchen）是指在一個集中的生產設施中，大規模生產標準化餐食，再配送至多個服務地點（如各長照機構、日照中心、居家個案）的商業模式。在長照膳食業，中央廚房是**業者邁向規模化的核心基礎設施**。

中央廚房的本質是「把廚房從機構裡拿出來，集中到一個地方做得更好、更安全、更有效率」。對業者而言，建立或採用中央廚房，是從「一家機構一家機構各別處理」的零散模式，升級到「集中生產、多點配送」的規模化服務模式的關鍵轉折。

---

## 中央廚房的商業優勢

| 面向 | 自有中央廚房（業者）| 分散生產（機構各自廚房）|
|---|---|---|
| **食材採購成本** | 集中採購，量大折扣，成本低 | 各自小量採購，價格高 |
| **品質一致性** | 標準化配方和 SOP，每批次品質穩定 | 各機構廚師技術參差，品質不一 |
| **食安管理** | 集中一個場所管理 GHP/HACCP，稽查效率高 | 多處場所，稽查負擔分散 |
| **特殊質地生產** | 投資商用均質機，IDDSI 測試合規 | 一般廚房難以做到精確質地控制 |
| **人力效率** | 集中生產效率高，人均產能高 | 各機構廚師效率各異 |
| **拓展能力** | 增加配送路線可擴展服務範圍 | 要服務新機構就要新廚房 |

## 自建中央廚房的投資評估

### 初始投資規模（概估）

| 項目 | 費用估算 |
|---|---|
| **廚房場地找尋和裝修** | 200-500 萬元（依規模）|
| **基本廚房設備**（火爐、冰箱、洗碗機等）| 150-300 萬元 |
| **商用均質機**（如 Vitamix 商業版）| 5-15 萬元/台 |
| **冷凍設備**（急速冷凍機）| 50-150 萬元 |
| **HACCP 認證費用** | 20-50 萬元 |
| **冷鏈物流設備** | 依配送範圍，100-500 萬元不等 |
| **總計初估** | **500 萬 - 1,500 萬元** |

## 中央廚房的食安管理要求

中央廚房作為食品製造場所，必須符合更嚴格的食安規範：

### 🔴 必備（法規要求）
- **GHP（食品良好衛生規範）稽查合格**：縣市衛生局定期稽查，合格記錄是 2024 年後外包合約的法定要求
- **食品工廠登記**：製造量超過一定規模需向衛福部登記

### 🟡 強烈建議（競爭優勢）
- **HACCP 認證**：完整的危害分析和關鍵控制點管理系統
- **留存檢體制度**：每批次食物的樣品保存至少 48 小時，整批留存 72 小時

### 🟢 加分項目（市場差異化）
- **ISO 22000 認證**：HACCP 的升級版，國際通用
- **Eatender 認證產品**：業者旗下特定產品取得 Eatender 標章

## 中央廚房的溫控物流挑戰

中央廚房集中生產後，食物必須透過**冷鏈物流**安全地送抵各配送點（詳見[冷鏈技術](冷鏈技術.md)和[物流宅配](物流宅配.md)詞條）：

- **熱食配送**：保持在 60°C 以上（危險溫度帶 5-60°C）需使用保溫箱
- **冷凍配送**：維持 -18°C 以下，需冷凍車或乾冰冷藏運輸
- **冷藏配送**：維持 0-4°C，適合隔天取用的新鮮餐食

**配送範圍決定商業模式**：
- 北台灣配送：30-60km 半徑，可以每日配送熱食或冷藏食品
- 全台配送：通常以冷凍食品（可常溫儲存後再冷凍配送）為主

## 「借用中央廚房」的輕資產模式

對於資本有限、剛起步的長照膳食業者，自建中央廚房的 500 萬元以上投資是個高門檻。一個替代選項是**委託代工（OEM）**：
- 找已有 GHP 合格中央廚房的業者代工生產，自行貼牌
- 優點：零投資進入市場，快速驗證商業模式
- 缺點：利潤空間受壓縮，質地控制依賴代工方
