## 台灣長照相關的主要公/協會

台灣長照領域有多個主要的公協會，業者加入或積極參與這些組織，是建立業界人脈、取得政策資訊、提升行業聲望的重要管道：

### 1. 台灣長期照護專業協會（TALA）

- **性質**：非營利的長照專業人員協會
- **主要功能**：舉辦長照培訓課程、發布長照政策分析、推動長照標準化
- **對業者的意義**：加入 TALA 的業者，更容易接觸到機構護理長和照護長，是 B2B 人脈建立的重要場所
- 🌐 [tala.org.tw](https://tala.org.tw)

### 2. 台灣護理之家暨安養機構協會

- **性質**：護理之家和養護機構的業者公會
- **主要功能**：代表機構業者向衛福部反映政策訴求、舉辦評鑑準備和法規說明研討會
- **對膳食業者的意義**：了解機構端的採購需求和政策壓力的最佳管道，也是和多家機構院長同時接觸的展覽和活動平台

### 3. 台灣食品安全學會

- **性質**：食品安全學術和實務的跨界組織
- **主要功能**：推動食品安全教育、發布食品安全研究
- **對膳食業者意義**：了解最新食安規範和學術趨勢

### 4. 台灣吞嚥障礙學會

- **性質**：SLP 和跨域吞嚥照護專業人員的學術協會
- **主要功能**：推動 IDDSI 在台灣的落實、發布吞嚥障礙臨床指引
- **對膳食業者意義**：和 SLP 社群建立關係的最佳機會，參與年會和工作坊可以接觸到業界頂尖的吞嚥照護專業人員

## 重要政府主管機關

| 機關 | 主管業務 |
|---|---|
| **衛福部長照司** | 長照 2.0 政策、機構評鑑 |
| **食藥署（TFDA）** | 食品安全法規、GHP 和 HACCP 合規 |
| **農業部農村發展及水土保持署** | Eatender 銀髮友善食品標章、食農教育 |
| **勞動部** | 長照人員薪資和勞動條件 |
| **原住民族委員會** | 原住民族長照特殊需求 |

## 業者參與業界活動的策略

1. **展覽參展**：「台灣國際健康展（Health Taiwan Expo）」、「長照博覽會」——直接接觸機構採購人員和潛在合作夥伴
2. **研討會發表**：在長照相關研討會中分享「業者的 IDDSI 執行經驗」或「B2B 膳食服務的成效案例」，建立業界聲望
3. **加入工作小組**：部分公協會有「食安工作小組」或「膳食照護委員會」，加入可以參與政策建議的擬定，同時和政府官員建立關係
