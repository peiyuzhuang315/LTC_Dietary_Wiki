# 機構評價（LTC Facility Assessment / Evaluation）

> **分類**：政策/法規 ｜ 機構管理 ｜ 業者關係  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：家屬、機構管理者、長照業者

---

## 什麼是長照機構評價？

**長照機構評價**（長照評鑑、機構評鑑）是台灣衛福部長照司對全國長照機構（護理之家、養護機構等）定期進行的**品質稽核和評核機制**，通過評核結果來管理機構的照護品質，並對社會大眾公開評核等第（讓家屬在選擇機構時有參考依據）。

評鑑結果通常分為「**優等**、**甲等**、**乙等**（需限期改善）、**丙/不合格**（可能被要求停止收住）」等等第。機構的評鑑等第，對家屬選擇機構的決策有重要影響，也直接關係到機構的聲譽和住民入住率。

---

## 台灣長照評鑑的核心架構

**[✅ Fact-Checked]** 衛福部長照評鑑主要看的面向：

| 評鑑面向 | 主要查核重點 |
|---|---|
| **行政和組織管理** | 人員配置合規、文件記錄、訓練計畫 |
| **護理照護品質** | 傷口照護、用藥管理、感染控制 |
| ****生活照護品質** | 日常活動、社交參與、尊嚴照護 |
| ****膳食照護** | 住民營養評估、個人化飲食計畫、食安管理 |
| **環境安全** | 設施安全、消防、輔具管理 |
| **住民和家屬滿意度** | 問卷調查、申訴記錄 |

---

## 膳食照護在評鑑中的重要性

**膳食照護是長照評鑑中的重要評分項目**，主要查核：

1. **營養評估是否定期執行**：住民是否至少每 3-6 個月進行一次 MNA 或同等工具的营養評估，高風險住民是否有縮短評估週期？
2. **個人化飲食計畫**：每位住民是否有書面飲食計畫，包含 IDDSI 等級和特殊飲食限制（低鈉、低磷、糖尿病飲食等）？
3. **食安管理文件**：是否有完整的留存檢體記錄、食材驗收記錄、食品溫度管理記錄？
4. **住民進食量追蹤**：是否有記錄住民的每日進食量，高風險住民是否有更密集的追蹤？

## 評鑑對業者的商業意義

業者在關係維護中，**主動協助機構「評鑑準備」是最有效的客戶留存策略**：

### 評鑑前 6 個月

- 提供機構「評鑑準備自我檢核表」（膳食相關部分），讓護理師知道評鑑時需要準備哪些文件
- 確認留存檢體記錄完整
- 提供 MNA 評估數據的整理和匯總格式，讓護理師可以直接插入評鑑報告

### 評鑑當天支援

- 業者派人到場（或提供電話即時支援），回答評鑑委員關於供餐業者資格、IDDSI 合規、食安管理等問題
- 提供業者的 GHP 合格記錄、HACCP 認證文件，供委員審查

### 評鑑後跟進

- 如果評鑑結果有需要改善的膳食照護問題，業者主動協助規劃改善方案，並在 3-6 個月內追蹤成效

---

## 家屬如何使用評鑑資訊選擇機構？

台灣衛福部長照評鑑結果公開於「**長照專區**」網站，家屬可以搜尋特定機構的評鑑等第：
- 優等機構：照護品質獲得最高肯定，在同等收費下優先考慮
- 甲等：品質良好的一般標準
- 乙等（限期改善中）：可進一步詢問機構改善進展，謹慎評估
- 有申訴記錄的機構：需要更仔細詢問
