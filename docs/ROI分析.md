# 詞條：ROI 分析（Return on Investment）
> **所屬面向**：市場分析面向 | **改寫日期**：2026-03-18

---

「你的產品很好，但我們預算有限。」——這是院長說得最多的一句話，也是業者一遇到就啞口無言的一道牆。問題不在院長不想買，而在業者**沒有辦法讓院長算清楚「不買的代價」**。

**[ROI分析](ROI分析.md)** 是業者打破這道牆的唯一鑰匙。長照膳食的 ROI 不只是「省了多少錢」，而是「防止了多少風險、提升了多少收入潛力」——把這四個維度說清楚，合約的對話就會從「要不要買」轉換成「怎麼規劃預算」。

### 四視角 ROI 論述框架

#### 視角①：外包省下的人力與設備成本

以 50 床機構為例：

| 項目 | 機構自煮 | 業者外包 |
|---|---|---|
| 廚師 2 名（含勞健保）| 年 840,000 元 | 0 |
| 設備折舊 | 年 30,000 元 | 0 |
| 水電 | 年 24,000 元 | 0 |
| 食材（無規模議價）| 年 240,000 元 | **含在外包費** |
| 外包費 | — | 年 420,000-510,000 元 |
| **合計** | **年 ≈ 1,134,000 元** | **年 ≈ 420,000-510,000 元** |

**結論**：外包每年可省 **60-70 萬元**（50 床規模）

#### 視角②：吸入性肺炎住院的防止成本

**[⚠️ 待查證]** 台灣每次吸入性肺炎住院的完整費用（含住院、後續復健、照護中斷損失）：業界估計 **3-8 萬元/次**，需健保資料庫進一步確認。

> 邏輯：IDDSI 合規膳食 + SLP 評估 → 若降低吸入性肺炎發生率哪怕 30%，以每季 2 次事件估算，一年可避免 0.6 次住院 = 省下 1.8-4.8 萬元

#### 視角③：評鑑合格對機構收入的保全

評鑑「不合格者」面臨停辦壓力、床位收入中斷——若一家 80 床機構月收費平均 3.5 萬元，停辦一個月損失 **280 萬元**。業者協助通過評鑑膳食項目，直接保住院長最核心的商業利益。

#### 視角④：住民滿意度提升帶動口碑入住率

長照機構的自費住民（不依靠政府補助）是**最高毛利的收入來源**。住民口碑 → 家屬推薦 → 自費入住率提升 → 院長最終財務表現改善。

### 業者的 ROI 說服工具

建議業者準備一份 **Google Sheets ROI 試算表**，讓院長在洽談時現場輸入床位數，即時看到自煮成本 vs 外包成本 vs 防止住院節省成本的對比數字。這是最有說服力的銷售工具。
