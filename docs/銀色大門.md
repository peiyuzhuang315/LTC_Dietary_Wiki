# 銀色大門（Silver Door）

> **分類**：業者業者 ｜ 服務模式 ｜ 社會企業

## 什麼是銀色大門？

**銀色大門**（Silver Door）是台灣一家以**居家送餐服務為核心**的社會型企業，由中興大學食品暨應用生物科技學系教授陳炳輝（現任總經理）等人共同創辦，總部位於台中，以北台灣和中台灣為主要服務區域。銀色大門的商業模式在台灣長照産業中多被視為一個**「送餐 + 關懷訪視」結合型居家服務」的代表性案例**。

**「銀色大門」這個名字的意涵**：銀（Silver）代表銀髮族，大門（Door）代表「直接抵達長者家門口的服務」——這正是銀色大門區別於一般食品批發業者的核心定位：不只是送食物，而是**以食物為介面，提供居家關懷**的整合服務。

---

## 銀色大門的服務模式

### 1. 居家送餐（核心服務）

銀色大門的主要客群是：
- **居家長照個案**：透過長照 2.0 補助或自費接受送餐服務的失能長者
- **社區獨居長者**：沒有家人協助、自行備餐困難的老年人
- **日照中心配餐**：部分日照中心委托銀色大門提供每日午餐

食物型態：以**常溫或冷藏便當**為主，依個案需求提供一般餐或軟質調整餐。

### 2. 關懷訪視（服務亮點）

銀色大門的差異化核心在於：**每次送餐都是一次訪視**。

送餐志工或員工在送餐後填寫「**關懷日誌**」，記錄：
- 長者今日的精神和外觀狀態
- 長者是否有不尋常的狀況（跌倒痕跡、意識改變）
- 長者今天的進食量（大約幾成）

這些資料定期彙整給家屬或合作機構，成為「居家長者健康哨兵系統」的一部分。銀色大門的服務人員被訓練為「非正式的第一線健康觀察者」。

**[✅ Fact-Checked]** 銀色大門服務現況：
- 主要服務區域：台中、彰化、台北、新北（部分區域）
- 服務量：每月送餐量超過數千人份
- 已進入長照 2.0 特約服務提供單位名單（部分縣市）

## 銀色大門的商業模式分析

| 收入來源 | 說明 |
|---|---|
| **長照 2.0 補助收入** | 透過特約服務提供單位資格，服務符合資格個案時可請款政府補助 |
| **個案自費收入** | 不符合補助資格或選擇自費加購的個案 |
| **機構 B2B 合約** | 日照中心、社區據點的定期配餐外包合約 |
| **公益捐款** | 作為社會型企業，部分低收入長者的服務費用由捐款支持 |

## 銀色大門模式對業者的啟示

銀色大門的成功提供了台灣長照膳食業者一個重要的**差異化路徑示範**：

1. **「食物只是敲門磚」思維**：送餐本身不是終點，而是接觸居家長者最自然、最低門檻的服務介面。通過送餐建立的信任關係，可以拓展為更多附加服務（如輔具協助、家屬諮詢）。

2. **關懷記錄系統的商業價值**：每次訪視記錄累積成「健康時間序列數據」——這個數據對家屬、社工、甚至保險公司都是有價值的資訊資產。

3. **社會影響力 = 品牌資本**：銀色大門在公益形象上的投資，讓它在政府採購、媒體曝光、合作夥伴关係建立上，有一般商業業者難以取得的信任加分。

## 居家送餐的長照 2.0 補助資格

**[✅ Fact-Checked]** 依長照 2.0 規定，送餐服務補助條件：
- **服務對象**：ADL 失能等級 2-8 級（需通過長照評估）
- **補助額度**：依住民失能等級，每月有一定的點數額度，送餐服務占用點數
- **申請方式**：聯絡各縣市長照管理中心（電話 1966），由長照管理師評估後安排
> 來源：[1966.gov.tw](https://1966.gov.tw)
