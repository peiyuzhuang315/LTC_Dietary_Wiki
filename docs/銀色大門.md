# 詞條：銀色大門（Silver Gate）
> **所屬面向**：業者詞條 | **改寫日期**：2026-03-18

---

「把送餐員升級成照顧安全網的一部分」——這句話是銀色大門最核心的商業主張，也是它為什麼不只是一家送餐公司。

**[✅ Fact-Checked]** 銀色大門（Silver Gate Elderly Care）基本資料：
- **執行長**：孫士姍
- **共同創辦人**：范曦、張鈺梅
- **服務範圍**：**11 縣市**
- **服務量**：每月超過 **3 萬人次**
- **物流合作**：Uber、pandago、愛鄰騎士 iLink
- **核心技術**：SaaS 路徑最佳化系統
- **重要功能一**：送餐員提交即時「**關懷日誌**」（住民狀況、情緒、進食量記錄）
- **重要功能二**：住民未到家自動警報機制（降低獨居長者意外風險）
> 來源：[sme.gov.tw](https://sme.gov.tw)

### 三個讓銀色大門超越「普通送餐公司」的設計

| 設計 | 商業意義 |
|---|---|
| **SaaS 路徑最佳化** | 每降低 10% 送餐時間 = 在不增加人力的情況下服務更多住民 = 毛利改善 |
| **關懷日誌提交** | 政府、家屬、機構同時需要這份數據 → 銀色大門成為不可替代的資訊節點 |
| **未到家自動警報** | 解決獨居長者最大的安全疑慮 → 讓家屬放心、讓政府埋單（長照給付激勵）|

### 業者和銀色大門的合作機會

對長照食品業者而言，銀色大門不只是競爭者觀察對象，更是**冷鏈食品的最佳通路合作夥伴**。可能的合作切入點：
1. **食材供應合作**：銀色大門送的餐需要穩定上游食材，有 GHP / HACCP 認證的業者是優先合作對象
2. **白牌代工**：若銀色大門擴大特殊質地餐的需求，擁有 IDDSI 等級食品的業者可成為代工夥伴
3. **補充品搭售**：在送餐服務中附贈業者的蛋白質補充品試用包（雙品牌行銷）

**[✅ Fact-Checked]** 物流成本在居家送餐模型中佔整體服務成本約 **40%**——這也是銀色大門以「路徑最佳化 SaaS」為核心競爭力的根本邏輯。
