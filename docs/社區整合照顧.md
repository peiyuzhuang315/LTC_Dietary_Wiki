# 社區整合照顧（Community-Based Integrated Care）

> **分類**：政策背景 ｜ 服務模式 ｜ 居家照護  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：家屬、社工、長照業者

---

## 什麼是社區整合照顧？

**社區整合照顧**（英文：Community-Based Integrated Care，或 Aging in Place）是一種以「**讓長者留在熟悉的社區環境中生活」**為核心理念的照護模式，相對於「機構集中照護（把長者送到護理之家、養老院）」的傳統做法。

社區整合照顧的基本主張是：**老年人在自己成長和生活的社區中，有更高的心理福祉、生活品質和社會連結感**——這一點在研究上已有相當充分的支持。台灣和全球主流的長照政策方向，都在從機構照護轉向以社區和居家照護為重心。

---

## 台灣「在地老化」的政策框架

**[✅ Fact-Checked]** 台灣長照 2.0 的核心設計，就是以「在地老化」為指導原則，建立覆蓋全台社區的 **ABC 三級據點系統**（詳見[[長照2.0]]詞條）：
- **A 級**：社區整合型服務中心，負責個案管理
- **B 級**：複合型服務中心，提供日間進行照護服務
- **C 級**：超過 4,000 個巷弄長照站，提供最基層的社區關懷和共餐

## 社區整合照顧的六大服務面向

| 服務面向 | 說明 | 舉例 |
|---|---|---|
| **居家照護** | 照服員到家提供日常生活協助（沐浴、如廁、備餐）| 長照 2.0 居家服務 |
| **日間照護（日照）** | 白天提供照護和活動，晚上回家 | 日照中心 |
| **社區共餐** | 定期在社區據點共享一餐，維持社交功能 | C 級巷弄長照站共餐 |
| **喘息服務** | 讓主要照護者（家屬）有短暫休息的機會 | 短期入住機構或居家喘息 |
| **健康促進** | 社區體操、口腔運動、認知訓練課程 | 社區健康活動 |
| **居家護理** | 護理師到宅提供護理評估和換藥等醫療服務 | 長照 2.0 特約護理師 |

## 社區整合照顧中膳食的核心角色

在社區照護模式中，膳食是最基本、最高頻的照護需求之一：
- 居家個案每天需要 2-3 餐，但照服員上門服務時間有限（通常 1-2 小時/次），備餐服務覆蓋率有限
- **外部送餐服務**（銀色大門等）填補了照服員不在家時的飲食缺口
- **共餐活動**提供了超越食物本身的社交功能——研究顯示共餐的長者有更好的認知維持效果

## 業者在社區照顧中的服務機會

| 業者機會 | 說明 |
|---|---|
| **C 級據點供餐外包** | 超過 4,000 個 C 級據點定期需要共餐食材，是中小型供餐業者的穩定訂單來源 |
| **居家送餐合約** | 作為縣市長照管理中心的特約送餐業者，服務符合資格的居家個案 |
| **日照中心午餐合約** | 日照中心通常需要每日午餐，是穩定的 B2B 合約 |
| **社區衛教合作** | 在社區活動中提供食安衛教，提高品牌曝光度 |
