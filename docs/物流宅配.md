# 物流宅配（Logistics and Home Delivery for LTC Food）

> **分類**：經營/營運 ｜ 通路管理 ｜ 冷鏈技術  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：長照業者、物流管理者

---

## 長照食品物流的特殊性

長照食品的物流和一般食品物流有根本性的差異：受眾（老年失能長者）對食品的**安全性、到達時間、保存溫度**有更嚴格的要求，而食品本身（特別是質地調整餐和需要冷鏈的特殊食品）也有更嚴格的流通條件。

---

## 三種主要配送模式

### 1. 自有車隊配送（最高控制力）

- 業者自行購置配送車乐划，由自己的配送人員執行
- **優點**：時間和溫度完全自主控制；配送人員熟悉業者的服務文化（可以兼顧關懷訪視功能）
- **缺點**：固定成本高（車輛購置、保險、人員薪資）；路線擴展困難
- **適合階段**：服務集中、路線固定、每日配送量已穩定的成熟業者

### 2. 第三方冷鏈物流（最靈活）

- 委託專業冷鏈物流公司（如統一速達冷凍、台灣宅配通 B2B）執行配送
- **優點**：靈活、可快速擴展配送範圍；不需要購置車輛
- **缺點**：溫度控制依賴第三方合規；每件配送成本較高；無法完成訪視功能
- **適合階段**：剛起步業者，或配送地點分散（如全台電商宅配）的全國型業者

### 3. 混合模式（最常見）

- 市區和固定路線使用自有車隊（效率高），偏遠地區或特殊訂單委外
- 業者可依路線密集度和成本效益，動態決定哪些路線自行配送

## 宅配冷鏈的溫度要求

| 配送類型 | 溫度要求 | 台灣主要服務業者 |
|---|---|---|
| **冷凍宅配**（-18°C 以下）| 運輸全程維持冷凍 | 黑貓冷凍宅急便、台灣冷鏈協會成員業者 |
| **冷藏宅配**（0-7°C）| 運輸全程冷藏 | 各縣市有專業冷藏配送業者 |
| **保溫配送**（> 60°C）| 熱食需維持 60°C 以上抵達 | 業者需自備保溫容器或保溫箱 |

## 配送效率最佳化工具

| 工具類型 | 功能 | 台灣可用工具 |
|---|---|---|
| **路線規劃軟體** | 自動計算最佳配送路線，減少燃料成本 | OptimoRoute、Route4Me、Google Maps API |
| **訂單管理系統** | 整合訂單、配送排程、住址管理 | 可結合 Shopify 等電商平台整合 |
| **溫度記錄設備** | 配送過程中自動記錄溫度變化，生成稽查文件 | 市售電子溫度記錄儀（Datalogger）|

## 最後一哩的特殊挑戰（Long-term Care Context）

長照食品的「最後一哩」有幾個一般電商不會遇到的挑戰：

1. **收件人可能行動不便**：長者無法自行到門口取件，需要配送人員將食品直接送到指定位置（廚房、冰箱放好）
2. **收件確認困難**：失智症或重度失能長者無法完成一般的收件程序（簽名、確認）
3. **緊急換貨需求**：住民吞嚥功能突然改變（如中風後），需要快速更換 IDDSI 等級，業者需要有靈活的換貨機制
