# 政策深解：長照膳食外包合規完整指南

> **適合閱讀者**：長照食品業者、機構管理者、品保人員  
> **閱讀時間**：約 12 分鐘  
> **關聯詞條**：[GHP食品良好衛生規範](../GHP食品良好衛生規範.md)、[HACCP認證](../HACCP認證.md)、[食品安全衛生管理法](../食品安全衛生管理法.md)、[機構評價](../機構評價.md)、[留存檢體](../留存檢體.md)

---

台灣長照膳食外包市場在 2024 年出現了一個重要的政策轉折點——衛福部正式公告，要求承接長照機構外包業務的膳食業者，必須持有有效的 **GHP（食品良好衛生規範）稽查合格記錄**。這不再是「加分」，而是**門檻**。

這篇文章從業者的角度，逐一拆解這個政策變化的實際影響，以及如何在最短時間內達到合規要求。

---

## 台灣長照膳食外包的法規架構

### 三層法規的關係

| 法規層次 | 名稱 | 對業者的要求 |
|---|---|---|
| **最高位法** | 食品安全衛生管理法（食安法）| 所有食品業者的基本遵循框架 |
| **技術規範** | 食品業者良好衛生規範準則（GHP）| 特定規模業者的衛生管理要求 |
| **業者自主** | HACCP 危害分析管制 | 自願申請的食安加強認證（大型業者常見）|
| **長照特別規範** | 長照機構膳食外包管理規定（2024）| 承接長照外包需具備 GHP 合格記錄 |

### 2024 年外包規範的核心要求

**[✅ 已生效]** 衛福部公告「長期照顧服務機構膳食外包管理規定」（2024 年 10 月），核心要求：

1. **業者需持有 GHP 稽查合格記錄**：由各縣市衛生局的食品安全稽查人員進行實地稽查，通過後核發合格記錄
2. **機構需留存業者資格文件**：機構采購前需索取、留存業者的 GHP 合格文件，並可在評鑑時展示
3. **業者需定期更新**：GHP 合格記錄不是一次性的，配合定期稽查維持合格狀態

## GHP 合規的六大面向

GHP 的稽查內容涵蓋六大面向，以下為業者應逐一確認的自我檢查清單：

### 1. 廠房和設施

- [ ] 食物製備區和非製備區有明確區隔（防止交叉汙染）
- [ ] 天花板、地板、牆面易清洗、無裂縫
- [ ] 窗戶有防蟲設計（紗網等）
- [ ] 足夠且適當的照明（工作區 ≥ 200 lux）
- [ ] 供水系統安全（每年至少一次水質檢驗）

### 2. 人員衛生

- [ ] 員工健康證明（每年 A 型肝炎、腸道疾病檢查）
- [ ] 工作期間正確穿戴帽子、口罩、手套
- [ ] 接觸食物前正確洗手 SOP（含洗手步骤海报）
- [ ] 員工生病時的替代措施

### 3. 食品處理和儲存

- [ ] 食材冷藏溫度 ≤ 7°C，冷凍 ≤ -18°C（有溫度記錄）
- [ ] 先進先出（FIFO）的食材管理原則
- [ ] 過期食材的識別和處理 SOP
- [ ] 生熟食分開儲存和處理

### 4. **留存檢體**（特別重要！）

依台灣食安規範，每餐食物必須**留存檢體 72 小時**：
- 每個菜色留存約 150-200 克
- 用有蓋容器密封，並標示**日期、餐別、菜色名稱**
- 存放於 0-7°C 冷藏
- 72 小時後確認無食安事件，才可丟棄

留存檢體是食安事件發生時的最重要證據，也是稽查人員必看的項目。詳見：[留存檢體](../留存檢體.md)

### 5. 害蟲管制

- [ ] 有定期的防蟲防鼠合約（外包給除蟲公司）
- [ ] 保存定期除蟲記錄
- [ ] 零發現原則：廠房內不允許出現任何害蟲跡象

### 6. 清洗和消毒

- [ ] 有書面的設備清洗頻率和方法 SOP
- [ ] 使用食品級消毒劑（濃度記錄）
- [ ] 定期環境微生物抽樣檢驗記錄

## HACCP 和 GHP 的關係：哪個更重要？

| | GHP | HACCP |
|---|---|---|
| **台灣長照外包的要求** | ✅ **必備**（2024年新規）| 加分，非強制 |
| **取得難度** | 中（稽查通過即可）| 高（需完整文件系統）|
| **費用** | 以稽查費為主（縣市政府）| 專業輔導費 5-30 萬元不等 |
| **更新頻率** | 每年例行稽查 | 3 年一次重新驗證 |
| **商業加分** | 基本要求 | 大型機構採購顯著加分 |

**建議路線**：
- **現在**：立即確保 GHP 合格（合規門檻）
- **一年內**：以 HACCP 作為下一步認證目標（競爭優勢）

## 業者合規路線圖（12 個月計畫）

### 第 1-2 個月：現況評估

1. 聯繫各縣市衛生局（食品藥物稽查科），詢問 GHP 稽查申請流程和排程
2. 進行廠房「自我稽查」（可以使用衛福部公開的 GHP 稽查表格作為對照）
3. 識別最主要的不合格項目

### 第 3-4 個月：硬體和 SOP 改善

1. 修繕廠房硬體（最優先：溫度計錄器、食材標示系統、留存檢體冷藏區）
2. 建立書面 SOP 文件（每個程序都需要書面化）
3. 開始員工訓練（GHP 基礎概念、洗手 SOP、溫度記錄填寫）

### 第 5-6 個月：試稽查和申請

1. 若有顧問資源，可邀請食安顧問進行模擬稽查
2. 向衛生局申請正式稽查
3. 正式稽查通過後，取得合格記錄

### 第 7-12 個月：維持合格 + 準備 HACCP

1. 建立每月例行的 GHP 自我稽查記錄
2. 開始 HACCP 危害分析文件準備（HACCP 計畫書的撰寫）

---

## 評鑑情境解析：衛生局來稽查，重點看什麼？

根據業者的實務經驗，食安稽查人員到場時最優先看的項目：

1. **留存檢體**（是否有？是否標示正確？是否在期限內？）
2. **冷藏冷凍溫度記錄**（連續 30 天的溫度記錄表）
3. **員工健康證明**（全員有效期限是否在效）
4. **食材進貨管理**（進貨日期標示、過期品是否立即下架）
