# 詞條：食品安全衛生管理法（Food Safety and Sanitation Act）
> **所屬面向**：政策/補助面向 | **改寫日期**：2026-03-18

---

在長照膳食業，「食安法」不是業者偶爾才要查的法條，而是每天都在影響你的商業行為的法律框架。一個「誇大功效」的業配廣告、一份「標示不完整」的食品標籤、一次「未達標準的留存檢體」——每一個疏失都可能讓業者面臨食安法的罰則。

**[食品安全衛生管理法](食品安全衛生管理法.md)**（簡稱：食安法）是台灣規範食品產業的最高層法律，2014 年後歷經多次重大修正（尤其是 2014 年大統長基油品風波後的全面強化），目前版本的重點罰則已達「最高 2 億元罰款 + 刑事責任」的嚴格程度。

### 長照膳食業者必知的六條高風險條文

| 法條 | 重點要求 | 違反後果 |
|---|---|---|
| **第 21 條** | 食品標示應完整（名稱、重量、成分、過敏原、保存條件、保存期限）| 可罰 3-15 萬元，情節重大最高 300 萬元 |
| **第 22 條** | 過敏原標示義務（11 大類：奶、蛋、花生、堅果、小麥、魚類、蝦蟹、大豆等）| 可罰 3-300 萬元 |
| **第 28 條** | 禁止不實廣告（不得聲稱未取得之功效）| 可罰 4-400 萬元 |
| **第 30 條** | 食品輸入須申報，不得逃漏申報（進口食材管理）| 可罰 6-1200 萬元 |
| **第 47 條** | 違規宣稱健康食品認證功效 | 最高 **2 億元**罰款 |
| **第 49 條** | 重大食安事件（攙偽摻假）| 七年有期徒刑以下 |

**[✅ Fact-Checked]** 2014 年食安法修正後，台灣食品業者面臨「連帶責任」機制：若下游業者使用了上游提供的問題食材且應知而不知，仍可能面臨處罰。這讓供應商合規稽查（KYS, Know Your Supplier）成為長照膳食業者的商業必修課。
> 來源：[consumer.fda.gov.tw食藥署](https://consumer.fda.gov.tw)

### 三個業者最常陷入的法律灰色地帶

1. **「高蛋白配方」不等於「適用於肌少症患者」**——除非有健康食品認證，廣告上不得使用「治療」「預防」等醫療聲稱
2. **「銀髮族適合」暗示功效**——廣告上特定訴諸銀髮族的健康聲稱，若無認證支撐，也在法律模糊地帶
3. **電商平台產品頁面的描述**——KOL 業配文和購物平台商品說明，業者同樣負有連帶責任
