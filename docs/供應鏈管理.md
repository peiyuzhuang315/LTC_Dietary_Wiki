# 供應鏈管理（Supply Chain Management for LTC Food）

> **分類**：經營/營運 ｜ 食安管理 ｜ 企業競爭力  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：長照業者、物流管理者

---

## 什麼是長照食品的供應鏈管理？

**供應鏈管理**（Supply Chain Management，SCM）是指業者對**從食材採購到最終配送至機構或住民手中**的整個流程，進行系統性的規劃、執行、監控和優化的管理活動。

長照食品的供應鏈比一般食品業更複雜，因為：
1. **食品安全要求更嚴格**：服務對象（失能長者）的免疫力低，食安事故後的後果比一般消費者嚴重得多
2. **質地要求的可追溯性**：每批次食物必須可以追溯到「這份食物是否達到 IDDSI Level X 的要求」
3. **配送末端的特殊性**：配送到的「最後一哩」往往是各種需要特殊照護的機構環境

## 長照食品供應鏈的六個節點

```
農場/食材供應商 → 驗收/進料管制 → 生產/加工 → 品質管控 → 儲存/冷鏈 → 配送 → 機構端
```

### 節點 1：食材供應商管理

- 建立**合格供應商名冊**：只向有 GHP 合格記錄的供應商採購
- **定期稽查供應商**：每年至少一次到供應商廠房查驗
- **在地優先原則**：優先選擇台灣本地農產（符合 Eatender 在地比例要求，同時降低食材旅行距離）

### 節點 2：進料驗收

每批食材到貨時必須執行：
- **外觀檢查**：顏色、氣味、包裝完整性
- **溫度確認**：冷藏 < 7°C，冷凍 < -18°C（留存確認記錄）
- **重量/數量確認**：和訂單對比
- **標示確認**：保存期限、食材來源標示完整

### 節點 3：生產/加工

- 依 GHP 規範執行清潔和衛生管理
- 按 HACCP 計畫書執行關鍵管制點（CCP）的監控和記錄
- **留存檢體**：每批次食物製作完成後留存樣品（見[[留存檢體]]詞條）

### 節點 4：品質管控

- 對質地調整食品執行 IDDSI 叉壓/流速測試，并記錄結果
- 對完成品的溫度管控（熱食保溫、冷食冷藏）

### 節點 5：儲存

- 先進先出（FIFO, First In First Out）原則
- 生熟食分離儲存
- 確認儲存溫度設備的工作狀態（自動溫度記錄）

### 節點 6：配送到機構端

- 配送過程溫度記錄
- 機構端驗收簽收（包含溫度確認）
- 留存配送記錄

## 供應鏈的可追溯性（Traceability）

**[✅ Fact-Checked]** 食品安全衛生管理法要求食品業者具備**食品追溯追蹤（Traceback）能力**：一旦發生食安事故，可以快速追溯問題批次的完整供應鏈記錄（從哪個農場採購、在哪個設備製作、配送到哪些機構）。

建立可追溯系統的工具：
- **批次號碼管理**：每批生產品都有唯一的批次號，對應採購、生產、配送記錄
- **電子化記錄系統**：比紙本更難造假、更容易查詢
- **QR Code 追溯**：部分業者在產品包裝上放置 QR Code，機構人員掃描即可查看完整的生產和配送記錄
