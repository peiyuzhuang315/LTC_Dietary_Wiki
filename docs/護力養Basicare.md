# 詞條：護力養 Basicare（Basicare Nutrition）
> **所屬面向**：業者詞條 | **改寫日期**：2026-03-18

---

在台灣長照補充品市場，「護力養」這個名字幾乎等於「市場本身」——超過 **50% 的市占率**、超過 **800 家** 長照醫療單位合作、每個在長照場域工作超過三年的護理師都知道這個品牌。但很少人真正理解它的護城河是什麼。

**[✅ Fact-Checked]** 護力養（Basicare Nutrition International Co., Ltd.）基本資料：
- **董事長**：邱秋華（邱秋華女士）
- **護理顧問**：鍾誠珠
- **主力產品**：護力養 PLUS 雙蛋白高纖均衡配方
- **蛋白質含量**：**16%**（2021 年升級版）
- **調升項目**：新增維生素 D + 鈣（針對骨骼保健需求）
- **長照醫療單位市占率**：**超過 50%**
- **合作機構數**：超過 **800 家**
- **主要通路**：長照機構（B2B）+ Momo、Yahoo 電商（B2C）+ 藥局
- **定價**：每罐約 **36 元起**（零售參考）
> 來源：[basicare.com.tw](https://basicare.com.tw)、[ankecare.com](https://ankecare.com)

### 護力養的護城河不是配方，而是這三件事

| 護城河 | 具體內容 | 競爭者難以複製的原因 |
|---|---|---|
| **顧問式服務** | 到機構辦衛教、訓練照服員、提供住民評估報告 | 需要大量專業人員和時間投入 |
| **SMPF 認證** | 取得衛福部「特定疾病配方食品」核可，可在外包裝進行特定適用性標示 | 申請耗時（1.5-2年）+ 費用高 |
| **機構先行建立口碑，家屬後台 B2C 跟單** | 機構護理師推薦 → 家屬出院後繼續在電商訂購 | 需要多年的機構口碑積累 |

### 產品進化路徑（業者參考）

**[✅ Fact-Checked]** 護力養產品歷史：
- 初版：以均衡熱量和蛋白質為核心（蛋白質 ≈ 12-14%）
- **2021 年升級**：蛋白質提升至 **16%**，新增維生素 D 和鈣

這個升級不是偶然——它直接呼應了 ESPEN 2019 年新版老年人蛋白質建議量（1.2-1.5g/kg/day），以及台灣科學界對肌少症防治的研究熱潮。升級時機的精準判斷，本身就是對市場的深度洞察。
