## 原住民飲食文化在長照的重要性

**[✅ Fact-Checked]** 台灣原住民族現況：
- 台灣原住民族（16 個族群）人口：約 **58 萬人**（占台灣總人口約 2.5%）
- 原住民族的老年化速度高於一般族群，且健康照護需求有明顯的文化差異
- 部分縣市（花蓮、台東、南投、屏東、宜蘭）的長照服務中，原住民住民比例相當高
> 來源：原住民族委員會

**文化照護（Culturally Congruent Care）**在長照領域是一個被越來越重視的議題：長者在熟悉的文化和飲食中，能夠更有尊嚴、更有自主感地生活。對原住民長者而言，這意味著能夠吃到熟悉的族群食物、用族語交流。

## 台灣原住民飲食的特色食材

原住民族的傳統食材，許多在台灣一般市場較為少見，但具有高度的文化意義和部分的營養優勢：

| 食材 | 說明 | 文化意義 |
|---|---|---|
| **小米（Millet）** | 許多族群的主食或儀典食物 | 豐年祭等重要儀式不可缺少 |
| **地瓜（Sweet Potato）** | 平埔族、邵族等族群的歷史主食 | 日常食物的文化根源 |
| **野菜（山蘇、芋葉莖）** | 各族群都有採集野菜的飲食習慣 | 部落生活的連結 |
| **醃生豬肉（阿拜 Abai 等）** | 排灣、魯凱等族的傳統食物 | 祭典食物，不一定適合日常長照提供 |
| **飛魚（達悟族）** | 蘭嶼達悟族的核心食材 | 深度文化連結（飛魚祭） |
| **樹豆（Pigeon Pea）** | 布農族等族群常用 | 氨基酸豐富，對長者蛋白質補充有潛力 |

## 長照場域提供原住民文化飲食的挑戰

1. **食材取得**：許多原住民傳統食材（山蘇、野菜、小米）在都市的食材市場較難穩定供應
2. **料理方式的文化知識**：原住民傳統料理方式（醃製、炭烤、蒸煮）和一般長照廚房的技術有差異
3. **族群內的差異**：16 個族群的飲食文化有很大差異，不能用「原住民食物」一概而論

---

## 對業者的文化照護建議

1. **詢問和記錄**：入住評估時，主動詢問住民「您平時最喜歡吃什麼？有沒有什麼從小就常吃的食物？」——文化食物偏好的記錄，是個人化照護的起點
2. **主動引進在地食材**：花東地區的業者，主動和原住民部落農場建立食材供應關係（小米、樹豆、山蘇等），同時符合 Eatender 在地農產要求
3. **節慶飲食設計**：在豐年祭、布農打耳祭等重要節期，為機構的原住民住民準備有文化意義的食物，讓住民感覺「被記得」
