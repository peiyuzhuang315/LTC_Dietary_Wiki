# 詞條：長照 2.0（Long-term Care 2.0）
> **所屬面向**：政策/補助面向 | **改寫日期**：2026-03-18

---

「長照 2.0」是台灣長照政策史上最重要的一次升級——從 2017 年上路，把分散在各部會的老年照護資源整合進一個統一的服務體系，讓失能長者不再需要一家一家機構問服務（雖然現實情況還是沒那麼流暢）。

**[長照2.0](長照2.0.md)** 的核心架構設計圍繞三個層次：**A 單位**（社區整合服務中心，負責評估和轉介）、**B 單位**（複合型服務，如日照中心）、**C 單位**（巷弄長照站，最貼近社區的服務據點）。這個 A-B-C 架構的關鍵是：讓失能長者「在地老化、在社區就能得到服務」，而不是全部推進機構。

### 長照 2.0 的給付結構（膳食相關項目）

**[✅ Fact-Checked]** 長照 2.0 膳費給付架構：

| 服務類型 | 給付說明 | 住民自付比例 |
|---|---|---|
| **機構式照護膳費補助**| 依失能等級和收入補助，低收入戶補助較高 | 依評估結果，通常 **10-30%** 自付 |
| **居家送餐服務** | 需求者每餐補助（依縣市有所不同，約 **50-100 元/餐**）| 差額自付 |
| **日照中心午膳** | 多數日照中心每日午餐費用由機構向縣市政府申請 | 部分補助 |
| **共餐服務（C 單位）** | 補助社區關懷據點的共餐食材費 | 長者通常每餐只需 **30-50 元** |

**[⚠️ 待查證]**「長照 3.0」接棒計畫的完整膳費補助調整幅度：各子計畫補助細則尚在修訂，建議追蹤衛福部 2026 年第二季公告。

### 業者應掌握的長照 2.0 資訊管道

要追蹤長照給付變化，這三個資訊來源是業者的必讀清單：
1. **衛福部 1966 照護專線官網**（[1966.gov.tw](https://1966.gov.tw)）：政策異動第一手來源
2. **AnkeCare 創新照顧**（[ankecare.com](https://ankecare.com)）：媒體化整理，可讀性高
3. **台北國際照顧博覽會**（每年 10-11 月）：政策宣布的重要場合，常有最新補助消息
