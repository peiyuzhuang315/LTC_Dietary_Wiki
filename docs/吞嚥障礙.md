# 吞嚥障礙（Dysphagia）

> **分類**：照護需求 ｜ 臨床症狀 ｜ 膳食設計基礎  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：家屬、照服員、長照業者、醫護人員

---

## 什麼是吞嚥障礙？

**吞嚥障礙**（英文：Dysphagia，源自希臘文 dys「困難」+ phagein「吃」）是指一個人在吞嚥食物或液體的過程中，出現困難、疼痛、或食物誤入氣管的狀況。它不是單一的疾病，而是由多種原因造成的**症狀群**——就像「發燒」是症狀而非疾病，吞嚥障礙背後可能隱藏中風、巴金森氏症、失智症、老化等截然不同的病因。

在長照照護現場，吞嚥障礙是每位第一線工作人員都必須認識的核心概念，因為它直接決定了：
- 住民**今天的每一餐應該吃什麼質地的食物**
- 護送食物進入身體的過程中**有多高的誤吸風險**
- 不處理的話，接下來可能面臨**吸入性肺炎**的緊急住院

## 吞嚥的生理機制：一個你從沒注意過的精密動作

正常的吞嚥看似輕鬆，實際上是**人體最複雜的協調動作之一**，需要至少 30 條肌肉、5 對腦神經（第五、七、九、十、十二對）在不到 **1 秒鐘**內精確協作。吞嚥分為三個階段：

### 第一階段：口腔期（Oral Phase）
食物進入口腔後，舌頭和頰肌負責將食物與唾液混合、咀嚼，最終形成可以吞嚥的食團（Bolus）。老化會讓這個階段出現：
- **牙齒缺損**：咀嚼力下降，食物無法充分磨碎
- **唾液分泌減少**：食物難以潤濕，形成食團困難
- **舌壓力下降**：無法有效將食團推向咽喉

### 第二階段：咽喉期（Pharyngeal Phase）——最危險的關卡
食團抵達咽喉後，身體必須在 **0.5 秒內**同時完成兩件事：
1. **軟顎上升、關閉鼻腔通道**（防止食物嗆進鼻子）
2. **會厭軟骨下翻、覆蓋氣管開口**（防止食物進入氣管）

這個動作一旦延遲或失誤，食物就會**誤入氣管（誤吸，Aspiration）**。對年輕健康的人，誤吸可以透過強力咳嗽將食物排出；對失能長者，**咳嗽反射也已退化**，異物安靜地滑入肺部，引起吸入性肺炎——這也是為什麼臨床上稱之為「**靜默性誤吸（Silent Aspiration）**」。

### 第三階段：食道期（Esophageal Phase）
食物進入食道後，靠食道肌肉的蠕動推入胃部。這個階段的障礙（如食道狹窄、胃食道逆流）相對容易處理，對長照膳食設計的影響相對較小。

## 誰容易有吞嚥障礙？

吞嚥障礙可以發生在任何年齡，但在長照場域最常見的高風險族群包括：

| 風險族群 | 吞嚥障礙的主要機制 |
|---|---|
| **中風後患者** | 腦部控制吞嚥的神經通路受損，咽喉期協調失敗 |
| **巴金森氏症** | 神經退化導致肌肉僵硬、動作遲緩，咽喉期延遲 |
| **失智症（晚期）** | 認知功能下降導致無法啟動或完成吞嚥動作 |
| **頭頸癌治療後** | 放射線治療破壞吞嚥相關肌肉與神經 |
| **一般高齡老化** | 肌力退化、神經傳導速度減慢，所有階段效率下降 |
| **長期臥床者** | 廢用性肌肉萎縮，吞嚥肌群同步退化 |

> **重要觀念**：吞嚥障礙不等於「老了自然的事」。老**化確實增加風險，但吞嚥障礙是可以評估、可以介入、可以訓練改善**的——這就是言語治療師（SLP）存在的核心意義。

## 如何判斷家人有吞嚥問題？——家屬必看的 10 個警訊

你不需要是醫療人員，以下這些日常進食觀察，就是早期發現吞嚥問題的關鍵：

| 警訊 | 背後可能的問題 |
|---|---|
| ① 吃飯時**頻繁咳嗽**（尤其喝水時）| 液體誤吸最早期信號 |
| ② 飯後**聲音變沙啞**（「濕式嗓音」）| 食物殘留在咽喉，聲带被覆蓋 |
| ③ 吃飯時間**明顯拉長**（一餐超過 45 分鐘）| 咀嚼和吞嚥效率下降 |
| ④ 食物或液體**從嘴角流出** | 口腔期控制力退化 |
| ⑤ 每次吞嚥需要**多次才能完成**（分批吞）| 咽喉期肌力不足 |
| ⑥ **拒絕吃特定質地的食物**（特別是肉類或稀飯）| 住民已有「吃這個會不舒服」的自覺 |
| ⑦ 吃飯後**反覆發燒**（不明原因低燒）| 靜默性誤吸引發的輕微感染 |
| ⑧ **體重持續下降**（非刻意減重）| 因吞嚥困難而減少進食量 |
| ⑨ 喝水**嗆到幾乎每次** | 液體流速快，難以控制 |
| ⑩ 住院後突然不肯吃飯（常見於**急性病後**）| 急性病加速吞嚥功能退化 |

**如果你的親人出現上述 3 項以上警訊，建議主動向主治醫師或機構護理長提出吞嚥評估的需求**，由言語治療師進行正式評估。

## 吞嚥障礙的正式評估方式

### 臨床床邊評估（Clinical Bedside Evaluation）
由受訓的言語治療師在床邊執行，包含觀察口腔結構、測試不同質地食物和液體的吞嚥反應。優點：快速、無輻射；缺點：無法看到咽喉內部的完整動態。

### 電視螢光透視吞嚥攝影（VFSS，Videofluoroscopic Swallowing Study）
俗稱「吞鋇劑攝影」，是目前吞嚥評估的**黃金標準（Gold Standard）**。患者在X光透視下吞嚥混入硫酸鋇的食物，醫師可以即時觀察食物在咽喉的移動路徑，精確判斷誤吸的時機和嚴重程度。

### 纖維喉鏡吞嚥攝影（FEES，Fiberoptic Endoscopic Evaluation of Swallowing）
透過鼻腔插入細小的內視鏡，直接觀察咽喉的吞嚥情況。適合無法配合轉移至放射科的臥床住民。

## 照護端的處理：IDDSI 質地分級是關鍵語言

目前全球（包含台灣農業部 Eatender 系統）最廣泛使用的質地管理框架是 **[[IDDSI]]（International Dysphagia Diet Standardisation Initiative，國際吞嚥障礙飲食標準）**。

IDDSI 把食物分為 0-7 級（0 最稀、7 最需要咬合力），由 SLP 依評估結果「開立處方」，告知廚房和照服員「這位住民只能食用 X 級食物、使用 Y 級稠度的液體」：

| IDDSI 固體等級 | 名稱 | 外觀描述 | 適合族群 |
|---|---|---|---|
| Level 7 | 一般食物 | 無限制 | 無吞嚥問題者 |
| Level 6 | 軟質一口量 | 叉子邊緣可切斷，1.5cm 以下 | 輕度咀嚼困難 |
| Level 5 | 細碎濕軟食物 | 4mm 以下碎粒，需水分潤濕 | 中度咀嚼困難 |
| Level 4 | 泥狀食物 | 均質無顆粒，不會流動 | 重度咀嚼+輕度吞嚥困難 |
| Level 3 | 液狀食物 | 可從湯匙緩慢流下 | 需要流動性但仍有一定控制 |

| IDDSI 液體等級 | 名稱 | 流速描述 |
|---|---|---|
| Level 0 | 稀薄液體 | 一般清水 |
| Level 1 | 微稠液體 | 略比清水稠 |
| Level 2 | 低稠液體 | 類似牛奶 |
| Level 3 | 中稠液體 | 類似蜂蜜 |
| Level 4 | 高稠液體 | 類似布丁 |

> **重要**：讓液體變稠的工具稱為「[[增稠劑]]」。目前市面上主要有**澱粉類**和**膠體類（如黃原膠）**兩種，後者不受溫度影響，長照機構普遍採用。

## 吞嚥訓練與復健：退化不是終點

部分民眾誤以為「吞嚥退化了就沒辦法」——其實**言語治療師可以透過系統性訓練，讓部分住民恢復到更好的吞嚥等級**，甚至從鼻胃管（管灌飲食）成功脫管，重新經口進食。常見的吞嚥訓練方法包括：

- **舌壓訓練**（Iowa Oral Performance Instrument，IOPI）：增強舌頭推送食物的力量
- **Shaker 訓練**：平躺、頭部抬起訓練，強化舌骨上肌群
- **Masako 法**：咬住舌尖的同時吞嚥，訓練咽後壁肌群
- **溫暖刺激（Thermal-Tactile Stimulation）**：以冰鏡子刺激口腔前弓，誘發更快的吞嚥反射

> 訓練需由受訓的 SLP 指導執行，家屬和照服員協助重複練習。每日 10-15 分鐘的訓練，4-6 週可看到改善。

## 長照膳食業者的責任與機會

對長照膳食供應業者而言，吞嚥障礙是**既是風險也是差異化優勢**的核心議題：

**風險面**：如果業者供應的食物質地不符合住民的 IDDSI 等級處方，導致誤吸事件，業者在法律上面臨食安責任——「你說這是軟質食物，護理長說可以，但沒有 SLP 的正式評估」，不能免責。

**機會面**：能夠精確提供 IDDSI Level 4、5、6 等不同等級食物，並且與 SLP 協作出具「質地合規報告」的業者，在 B2B 機構採購中具有明顯的競爭優勢。這是大型連鎖團膳公司難以快速複製的「臨床整合服務能力」。

**建議行動**：
1. 讓研發人員取得 **IDDSI 工作坊認證**（台灣咀嚼吞嚥障礙醫學學會定期舉辦）
2. 在菜單設計上，每道菜標示對應的 **IDDSI Level**
3. 與合作機構的 SLP 定期確認住民的質地處方是否更新
