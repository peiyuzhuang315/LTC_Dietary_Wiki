# 📋 認證與法規

> 長照食品業者從基本合規到進階認證的完整路徑地圖。

---

## 🗺️ 認證路徑：從必備到加分

```
第一層（必備）：GHP 食品良好衛生規範
      ↓
第二層（進階）：HACCP 危害分析重要管制點
      ↓
第三層（國際）：ISO 22000 食品安全管理系統
      ↓
加分認證：Eatender 銀髮友善食品標章 / TQF / CAS
```
