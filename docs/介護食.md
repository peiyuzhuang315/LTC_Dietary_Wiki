# 詞條：介護食（Kaigo Shoku / Care Food）
> **所屬面向**：使用者/照護需求面向 | **改寫日期**：2026-03-18

---

「介護食」這個詞直接借自日文（介護食品 / かいごしょくひん），在台灣業界和學術圈廣泛流通，指的是「專為需要照護的人設計的，針對咀嚼和吞嚥困難進行質地調整的食品」。它和台灣慣用的「長照食品」「銀髮友善食品」在實質上高度重疊，只是起源語言不同。

**[介護食](介護食.md)** 在台灣市場的重要性，很大程度上來自日本市場的先驅效應：日本早台灣十多年面對超高齡化問題，介護食品市場規模達數千億日圓，建立了完整的 UDF（通用設計食品）等級體系，為台灣業者提供了最直接的商業參考藍圖。

### 台灣 vs 日本介護食市場對比

| 比較面向 | 日本 | 台灣 |
|---|---|---|
| **進入超高齡社會** | 1994 年（全球最早）| **2025 年 8 月** |
| **質地分級標準** | UDF 四等級（介護食品協議會）| IDDSI 0-7 級 + Eatender |
| **市場規模** | 數千億日圓（保守估計）| 數百億台幣（估算中）|
| **主力業者** | 明治、森永、QP、日清等大廠 | 漢典、護力養、格林等中型業者 |
| **認證體系** | UDF 自律認證 | Eatender（官方），IDDSI（臨床）|

**[✅ Fact-Checked]** 台灣介護食市場的關鍵推手：台灣咀嚼吞嚥障礙醫學學會（理事長：**王亭貴**，IDDSI 代表：**鄭千惠**），是台灣把日本「介護食」概念轉化為本地 IDDSI 臨床實踐的核心學術機構。

### 業者如何利用「介護食」的日本品牌光環？

台灣消費者對日本食品普遍有信任感——在 B2C 市場，若業者的產品說明中明確提到「符合日本 UDF 標準 / 對應 IDDSI X 級」，比純粹說「軟質食品」更能建立家屬的信任感。同時，出口日本市場的中長期目標，可以讓業者以「台灣本市場合規 → 日本市場出口」作為清晰的產品認證路線圖。
