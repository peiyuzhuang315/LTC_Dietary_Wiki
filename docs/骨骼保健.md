# 詞條：骨骼保健（Bone Health）
> **所屬面向**：產品/研發面向 | **改寫日期**：2026-03-18

---

骨骼保健在長照膳食的重要性，不只是「怕骨折」這麼簡單。**骨折是長照住民死亡率急劇上升的導火線**——老年人髖骨骨折後一年內死亡率高達 20-35%，即使存活也往往從此失去行動能力，直接升級為重度失能。預防骨折，從膳食角度來說，就是預防長照照護需求的惡化。

**[骨骼保健](骨骼保健.md)** 的膳食介入策略圍繞兩個核心營養素：**鈣（Calcium）** 和 **維生素 D（Vitamin D）**，兩者缺一不可——鈣是骨骼的建材，維生素 D 是讓鈣能夠被吸收的搬運工。

### 老年人骨骼保健的關鍵數字

**[✅ Fact-Checked]** 台灣骨質疏鬆症基金會 + 衛福部建議：

| 族群 | 鈣每日建議量（mg）| 維生素 D 每日建議量（IU）|
|---|---|---|
| 一般成人（19-50 歲）| 1,000 | 600 |
| **50 歲以上**| **1,200** | **800** |
| **長照住民、骨鬆高風險** | **1,200-1,500** | **1,000-2,000**（需醫師評估）|

**[✅ Fact-Checked]** 台灣 65 歲以上長者維生素 D 不足（< 20 ng/mL）的比例：**超過 50%**，與日照不足（長照住民室內化）和飲食攝取不足密切相關。
> 來源：[mohw.gov.tw](https://mohw.gov.tw)、台灣骨質疏鬆症基金會

### 業者的骨骼保健產品機會

| 產品切入點 | 具體做法 |
|---|---|
| **主食強化鈣** | 在主食粥品中添加乳酸鈣（不影響口感，標示可見）|
| **補充品強化維 D** | 護力養 2021 年升級版即新增維生素 D + 鈣；業者可參考 |
| **高鈣食材優先選用** | 優先採用豆腐、板豆腐、小魚乾、芝麻、深色蔬菜等高鈣食材 |
| **健康食品認證機會** | 「骨質保健」是衛福部核准的 13 大功效之一，適合蛋白質+鈣複合配方申請 |
