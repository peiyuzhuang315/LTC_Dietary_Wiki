# 詞條：機能性保健食品（Functional Health Food）
> **所屬面向**：產品/研發面向 | **改寫日期**：2026-03-18

---

「機能性保健食品」這個詞在市場上被用得很寬泛——廠商說的「機能性」，和政府法規認可的「保健功效」之間，往往存在一條法律上的鴻溝。業者如果沒有搞清楚這條線，很容易在廣告文案上踩雷。

**[[機能性保健食品]]** 在台灣有兩個層次的意涵：

**廣義（市場用語）**：任何在配方設計上加入特定機能性成分（如益生菌、膠原蛋白、葉黃素、魚油、大豆異黃酮）的食品，業界習慣稱之為「機能性食品」。這個詞**沒有法律定義**，廠商可以自由使用，但**不能因此聲稱特定功效**。

**狹義（法規層次）**：唯有取得衛福部「健康食品」核可的產品，才能合法聲稱上述成分對人體的保健功效。

### 長照膳食常見的機能性成分與市場定位

| 機能性成分 | 對長照族群的價值 | 合法聲稱前提 |
|---|---|---|
| **益生菌** | 腸道調理（便秘、菌相失衡）| 需取得「腸道菌叢改善」健康食品認證 |
| **天然型維生素 D3 + 鈣** | 骨骼保健（防骨折骨鬆）| 需取得「骨質保健」認證或維持在合法攝取量 |
| **EPA/DHA（魚油）** | 調節血脂、認知維持 | 需取得「調節血脂」健康食品認證 |
| **葉黃素** | 視力保健 | 需取得「視力保健」認證 |
| **植物提取蛋白（大豆分離蛋白）** | 肌少症蛋白補充 | 一般食品標示（不可聲稱治療）|
| **MCT 油** | 快速熱量補充、認知支持 | 一般食品標示 |

**[✅ Fact-Checked]** 台灣衛福部管轄的「健康食品」（小綠人標章）與一般機能性食品的核心差異：健康食品需完成動物實驗 + 人體試驗（或安全性評估），花費 100-300 萬元，耗時 1-3 年。機能性食品無此限制，但也無法做任何功效聲稱。
> 來源：[fda.gov.tw食藥署](https://fda.gov.tw)
