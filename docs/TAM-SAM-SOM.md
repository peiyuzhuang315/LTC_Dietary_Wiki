# 詞條：TAM / SAM / SOM 市場規模分析（Total / Serviceable / Obtainable Market）
> **所屬面向**：市場分析面向 | **改寫日期**：2026-03-18

---

「台灣長照膳食市場有幾百億商機！」——每家業者的簡報都這樣說，但這個數字是給誰的市場？你能實際觸及多少？這一季你實際能拿下多少？不分層思考的市場規模估算，往往只是精美的數字幻象。

**[TAM-SAM-SOM](TAM-SAM-SOM.md)** 是業務規劃和融資簡報中最基礎的三層市場拆分框架：

### 長照膳食業的三層市場框架

| 層次 | 定義 | 台灣長照膳食估算（2025 年）|
|---|---|---|
| **TAM**（Total Addressable Market）| 全球或全行業最大可能市場 | 全台老年人口膳食消費 × 365 天 ≈ **600-800 億台幣/年** |
| **SAM**（Serviceable Addressable Market）| 你能服務的目標市場（依地域、能力邊界）| 機構 + 居家送餐 + 補充品合計 ≈ **200-350 億台幣/年** |
| **SOM**（Serviceable Obtainable Market）| 你在 3 年內合理能拿到的份額 | 初創業者 ≈ **500 萬-3,000 萬台幣/年**（依起步規模）|

**[⚠️ 待查證]** 以上 TAM/SAM 估算為業界參考數字，來源整合多份報告，請業者針對特定細分市場（如：單純機構供膳 vs 含補充品 B2C）自行精算。

### 如何做出有商業說服力的市場規模簡報

業者在招商或申請補助時，最常被投資人/審計委員挑戰的就是「你的市場規模怎麼估來的」。幾個提升可信度的做法：

1. **從資料來源倒推**：衛福部統計的機構床數（16 萬床） × 每人每日膳食支出（150 元）×365 天 = 約 **87.6 億元/年**（機構式純膳食市場）
2. **SAM 依自身地理能力設定**：台北 + 桃園 + 新北三都的機構床數約佔全台 40%，可作為優先 SAM
3. **SOM 用現實客戶數推算**：第一年目標 5 家機構 × 每月合約金額 20 萬 × 12 個月 = 1,200 萬元（具體而非泛泛）
