# 消費者分群（Consumer Segmentation in LTC Dietary Market）

> **分類**：市場分析 ｜ 商業策略 ｜ 業者工具  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：長照業者、行銷人員

---

## 為什麼消費者分群對長照業者重要？

長照膳食市場的一個常見錯誤是：**業者把所有長照相關消費者當作單一族群**，用同樣的訊息、同樣的通路、同樣的產品試圖服務完全不同需求的客戶。這導致行銷費用分散、轉換率低、品牌定位模糊。

有效的消費者分群，讓業者能夠**聚焦資源在最有價值的目標客群**，用最能打動那個群體的語言和通路精準溝通。

## 長照膳食市場的消費者分群地圖

### Tier 1：機構採購人員（B2B 核心）

| 角色 | 院長 / 機構管理者 | 護理長 / 照護主任 |
|---|---|---|
| **決策權** | 最終採購決策者 | 具體需求提出者，院長信任的執行者 |
| **關心的是** | 成本效益、食安責任、整體服務可信度 | IDDSI 合規、住民滿意度、照護工作減輕 |
| **說服方式** | 成效數字（KPI）、業者資格認證、機構評鑑加分 | 試吃、照服員操作簡便、MNA 追蹤服務 |

### Tier 2：家屬（B2C + B2B 影響者）

家屬在兩個層次影響採購決策：
1. **影響機構採購**：當家屬抱怨「食物太難吃、太單調」，院長可能更換業者；當家屬稱讚，院長更傾向續約
2. **自行採購**：家屬為在家長者購買銀髮食品（電商）或為機構長者補充蛋白質補充品

| 子群體 | 特徵 | 溝通重點 |
|---|---|---|
| **責任型家屬** | 主動關心父母飲食，定期探訪 | 科學依據（IDDSI 標準、蛋白質含量）、機構食物品質透明度 |
| **遠距家屬** | 住在外縣市，無法常探訪 | 「遠距也能安心」的訊息，訂閱制宅配解决方案 |
| **節省型家屬** | 主要考量成本 | 平價但有基本保障（GHP 認證）的選項 |

### Tier 3：居家個案本身（部分有消費能力的長者）

- **活躍銀髮族（55-70 歲）**：有消費能力，主動購買預防性健康食品
- **輕度失能長者**：可以自行或透過照服員協助訂購，重視口感和多樣性
- **重度失能長者**：被動接受機構或家屬的採購決定，自己無法主動選擇

## 各分群的最佳觸及通路

| 消費者群體 | 最佳觸及通路 | 次佳通路 |
|---|---|---|
| 機構院長 | 業務直接拜訪、長照展覽、機構評鑑相關研討會 | 業者口碑推薦、LinkedIn |
| 護理長 | 業務拜訪（和院長一起）、衛教課程 | 醫療相關展覽 |
| 責任型家屬 | Facebook 社群（長照相關群組）、Google 搜尋 | 醫院衛教資料、LINE 推播 |
| 遠距家屬 | 電商（蝦皮/Momo）、Instagram、訂閱制平台 | Facebook 廣告 |
| 活躍銀髮族 | 電視廣告（長者收看比例仍高）、藥妝通路 | 社區活動贊助 |

---

## 分群行銷的實際執行建議

業者資源有限，**不需要同時服務所有分群**。建議的優先順序：

1. **先聚焦 B2B（機構院長 + 護理長）**：市場最大，每個客戶的 LTV（終身價值）最高
2. **再拓展家屬影響者**：透過機構長者的口碑和家屬口碑，建立市場聲譽
3. **最後開發 B2C 電商**：在品牌已有一定知名度後，B2C 電商通路的轉換率才會理想
