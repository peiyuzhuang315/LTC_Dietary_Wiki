# 詞條：口腔衰弱評估（Oral Frailty Assessment）
> **所屬面向**：團隊/專業面向 | **改寫日期**：2026-03-18

---

「口腔衰弱」（Oral Frailty）是一種進展性退化——但它的危險在於，發展到真正影響安全進食之前，往往有很長一段「灰色地帶」期，住民看起來還能吃，但口腔機能已在悄悄流失。**[口腔衰弱評估](口腔衰弱評估.md)** 就是在這段灰色地帶中及早辨識風險、提前介入的工具。

**[✅ Fact-Checked]** 日本東京大學飯島勝矢教授團隊提出的「口腔衰弱指標（Oral Frailty Index-8）」是目前最常被引用的口腔衰弱篩檢工具，包含 8 個問題，台灣已有多個學術單位和機構引進使用。
> 來源：台灣老年醫學期刊（多篇引用研究）

### 口腔衰弱的臨床篩檢指標（簡化版）

| 指標 | 評估方式 | 警戒信號 |
|---|---|---|
| **剩餘自然牙數** | 牙齒計數 | 少於 20 顆時風險上升 |
| **咬合力** | 咬合壓力計 or 主觀評估 | 咬不動堅硬食物 |
| **舌壓力** | 舌壓儀（Iowa Oral Performance Instrument）| 低於 20 kPa 為異常 |
| **嘴唇閉合力** | 脣力計 | 無法完整閉合（流口水）|
| **咀嚼功能** | 色素口香糖測試（咀嚼 50 次後顏色分布）| 顏色不均勻為異常 |
| **吞嚥功能** | 反覆唾液吞嚥測試（RSST）| 30 秒內少於 3 次為異常 |

**業者可提供的介入方案（對應評估結果）**：
- 舌壓低 → 建議「舌頭伸展 + 主動吞嚥」口腔運動方案
- 咀嚼力不足 → 調整至 IDDSI 5-6 等級軟質食物
- 吞嚥功能異常 → 轉介 SLP 進行完整 IDDSI 評估
