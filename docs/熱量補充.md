# 熱量補充（Caloric Supplementation）

> **分類**：照護需求 ｜ 營養介入 ｜ 產品設計  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：家屬、照服員、醫護人員、業者

## 熱量補充在長照的重要性

在長照場域，**熱量不足（Caloric Insufficiency）** 是和蛋白質不足並列的兩大核心營養問題。但熱量問題常常被忽視——因為「不夠胖」在台灣文化中反而有時被認為是「不錯的」，但對老年長者而言，**偏低的體重（BMI < 22）和快速的體重流失（每月 > 5%）是身體快速衰退的警訊，而非健康指標**。

**理解熱量和蛋白質的共生關係**：蛋白質補充品在熱量不足的情況下，效果會大打折扣——身體在缺乏熱量時，會優先把蛋白質用作能量，而不是用於維持肌肉。換句話說：**如果不先確保熱量充足，再多的蛋白質補充都是「燒掉」而非「建造」**。

## 老年人的熱量需求

**[✅ Fact-Checked]** 老年人熱量建議：
- 一般活動量的長者：每日 **1,600-2,000 大卡**（依體重和活動量）
- 計算公式（概估）：**30 大卡 × 體重（kg）**（正常體重長者）
- 失能臥床長者：可能只需要 **25-28 大卡 × 體重**（活動量大幅減少）
- 疾病恢復期或感染後：可能需要 **35 大卡 × 體重** 以上（分解代謝增加）

**台灣現況**：台灣長照住民的實際熱量攝取往往只達到需求量的 **70-80%**，體重過輕（BMI < 18.5）的比例在機構住民中超過 30%。

## 造成熱量不足的六大原因

| 原因 | 說明 |
|---|---|
| **食量縮減** | 老化導致胃容量縮小和荷爾蒙驅動的食慾下降 |
| **情緒和認知因素** | 失智症、憂鬱症降低進食意願 |
| **口腔和咀嚼問題** | 口腔疼痛或假牙不合，讓進食成為不舒服的體驗 |
| **藥物副作用** | 許多藥物（化療藥、某些抗生素）有噁心感副作用 |
| **疾病狀態** | 感染、炎症期的分解代謝升高（消耗熱量）|
| **食物不夠美味** | 食物口感差或缺乏多樣性，降低進食積極性 |

## 熱量強化的飲食策略

### 策略一：高熱量密度食材替代

用「同樣份量、更高熱量」的食材，在不增加進食份量的情況下提高整體攝取：

| 低熱量版本 | 高熱量密度替換 | 熱量增加 |
|---|---|---|
| 清湯 | 以牛奶或高湯取代部分水分 | +50-100 大卡/碗 |
| 蒸蛋 | 加入少量奶油或鮮奶油 | +30-60 大卡 |
| 白粥 | 加入花生醬或芝麻醬 | +100-150 大卡/匙 |
| 一般布丁 | 使用鮮奶油基底的高脂肪布丁 | +50-100 大卡 |

### 策略二：高熱量口服補充品（ONS）

口服營養補充品（Oral Nutritional Supplement，ONS）是最直接的熱量強化工具：
- **高熱量配方**（如 Ensure Plus）：每 200ml 提供 300 大卡（一般版只有 200 大卡）
- **脂肪添加劑**：MCT 油（中鏈三酸甘油酯）：每茶匙 40 大卡，無味、易溶於水，可加入任何食物

### 策略三：增加進食頻次

把原本的 3 餐改為 **5-6 次少量多餐**，每次份量小（住民更容易接受），但一天的總攝取量可以顯著提升。

## 熱量補充的品類機會

對長照業者而言，熱量補充品品類的市場機會包括：
- **高熱量均衡配方奶**（如護力養高熱量版本）
- **高熱量銀髮布丁**（兼具 IDDSI Level 4 質地和高熱量密度）
- **MCT 油添加包**（附在餐食中，家屬或照服員自行添加）
