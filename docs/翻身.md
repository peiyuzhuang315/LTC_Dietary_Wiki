# 臥床翻身：長期照護中不可或缺的關鍵技巧！

> **更新日期**：2024-07-25 ｜ **適合閱讀對象**：主要照護者（家屬、看護）、專業照服員、醫護人員

## 什麼是臥床翻身？

臥床翻身，在長期照顧領域中，指的是協助臥床或行動不便的長輩，定期改變身體姿勢的照護技巧。這項看似簡單卻至關重要的動作，對於無法自主活動的長輩而言，是維持身體健康、預防多種併發症的關鍵。當長輩長時間維持同一姿勢時，身體承受壓力的部位（如：薦骨、髖部、腳踝、肩胛骨、枕骨等骨突處）會因為血液循環不良，導致組織缺氧、壞死，進而形成壓瘡（俗稱褥瘡）。此外，長期臥床也容易造成肢體攣縮、肺部積痰、消化不良等問題。透過定時且正確的翻身，可以有效分散身體壓力、促進血液循環、維持皮膚完整性，並有助於呼吸道和消化道的順暢。當長輩因排泄物濕潤感到不適時，翻身不僅能讓長輩短暫離開潮濕環境，更能檢查並照護皮膚狀況，避免濕氣與壓力共同造成皮膚損傷。

---

## 為什麼臥床翻身如此重要？

臥床翻身對於長期照顧的家屬或機構而言，其重要性不言而喻。它不僅是基礎照護的核心環節，更是一種積極主動的預防措施，能有效避免嚴重併發症、顯著提升長輩的生活品質，並從長遠角度減輕照護負擔。

1.  **預防壓瘡（褥瘡）**：這是翻身最核心且首要的目的。壓瘡一旦形成，不僅會讓長輩承受極大痛苦，更可能導致傷口感染、敗血症，甚至危及生命。其治療過程漫長且複雜，耗費大量醫療資源與照護人力，對長輩和家庭都是沉重負擔。
2.  **預防肢體攣縮與關節僵硬**：長期臥床缺乏活動，肌肉容易萎縮、肌腱縮短，導致關節活動度受限，形成肢體攣縮，嚴重影響日後復健的潛力，甚至造成永久性失能。
3.  **促進呼吸道健康**：臥床長輩因重力關係，肺部底部容易積聚痰液，導致肺部擴張不全（肺塌陷）或吸入性肺炎。定時翻身有助於改變肺部受壓區域，促進痰液引流，維持呼吸道通暢，有效降低呼吸道感染風險。
4.  **改善血液循環與淋巴回流**：翻身能有效分散身體壓力，促進全身血液循環，減少水腫，確保組織獲得足夠的氧氣和營養，維持皮膚及其他器官的健康。
5.  **促進消化道功能**：臥床長輩常有腸蠕動減慢、便秘等問題。適當的翻身和體位改變，有助於刺激腸道蠕動，改善消化功能，減少腹脹不適。
6.  **提升心理舒適度與尊嚴**：長時間固定姿勢會讓長輩感到不適、煩躁，甚至產生被限制的無助感。翻身不僅能緩解身體不適，也能讓長輩感受到被關懷與尊重，提升心理上的舒適感和尊嚴。同時，翻身也是進行身體清潔、更換衣物或尿布的必要步驟，有助於維持個人衛生。
7.  **減輕長期照護負擔**：雖然翻身需要時間和體力，但相較於治療壓瘡或其他併發症所耗費的時間、金錢和精力，預防性的翻身顯然更為經濟有效。透過積極預防，能大幅降低醫療支出、減少照護者的身心壓力，從長遠角度實現更永續的照護。

## 翻身的頻率與時機：個別化評估是關鍵

翻身的頻率並非一成不變，需**根據每位長輩的個別狀況進行詳細評估與調整**。雖然「每兩小時翻身一次」是廣為人知的基本原則，尤其對於高風險的長輩，但這僅是一個參考點，實際操作時應考量多重因素。

**影響翻身頻率的關鍵因素包括：**

1.  **皮膚狀況與壓瘡風險等級**：
    *   **高風險長輩**：若長輩有糖尿病、營養不良、水腫、失禁、皮膚脆弱、已有壓瘡病史或正在使用類固醇等情況，其皮膚對壓力的耐受度較低，可能需要更頻繁的翻身，例如每1-1.5小時一次。
    *   **低風險長輩**：若長輩皮膚狀況良好，且能稍微自主活動或配合度高，則可維持每2-3小時翻身一次。
2.  **活動能力與意識狀態**：
    *   完全臥床、意識不清或無法自主翻身的長輩，需要嚴格遵守翻身計畫。
    *   部分能輕微移動肢體或能表達不適的長輩，可適度調整。
3.  **體重與體型**：
    *   過輕或過重的長輩，骨骼突出處或脂肪堆積處承受壓力更大，需更密切觀察並調整頻率。
4.  **使用的輔具**：
    *   雖然氣墊床、減壓坐墊等輔具能有效分散壓力，但它們並不能完全取代翻身。使用輔具的長輩仍需定時翻身，只是頻率可能可以稍微延長，但仍需密切觀察皮膚狀況。
5.  **長輩的舒適度與意願**：
    *   若長輩表達不適、煩躁，或觀察到其姿勢僵硬，應立即檢查並考慮翻身。
6.  **排泄物污染**：
    *   當長輩因大小便失禁導致皮膚潮濕時，應立即清潔並翻身，避免濕氣與壓力共同造成皮膚損傷。

**翻身時機的判斷與觀察重點：**

*   **定時檢查皮膚**：每次翻身時，都應仔細檢查長輩身體各骨突處的皮膚，特別是受壓部位是否有發紅、變色、水泡、破皮或溫度異常等壓瘡初期徵兆。若發現異常，應立即處理並調整翻身計畫。
*   **夜間照護**：夜間翻身對於預防壓瘡同樣重要，但需在確保長輩睡眠品質的前提下進行。可考慮使用減壓輔具，並在睡前、半夜、清晨等時段進行翻身，或根據長輩的睡眠週期調整。
*   **記錄與溝通**：詳細記錄每次翻身的時間、姿勢、皮膚狀況，並與其他照護者或醫護人員溝通，確保照護計畫的連貫性與有效性。

總之，翻身頻率與時機的決定，是一個動態且個別化的過程，需要照護者持續觀察、評估並彈性調整，以最大程度地保障長輩的健康與舒適。
