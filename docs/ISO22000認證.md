# ISO 22000 認證（Food Safety Management System）

> **分類**：認證/標章 ｜ 食安管理 ｜ 出口資格  
> **全名**：ISO 22000:2018 食品安全管理系統  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：長照業者、品保人員、出口業者

---

## 什麼是 ISO 22000？

**ISO 22000**（ISO 22000:2018 Food Safety Management Systems）是**國際標準化組織（ISO）**發布的食品安全管理系統標準，以 **HACCP 的危害分析原則**為核心，整合了品質管理（ISO 9001）的體系架構，形成一套**涵蓋整個食品供應鏈**的完整食品安全管理系統。

ISO 22000 和 HACCP 的主要差別：
- **HACCP**：聚焦在食品製造過程本身的危害管控
- **ISO 22000**：在 HACCP 的基礎上，加入了**組織管理系統**（相當於整個公司的食安管理制度，包含供應商管理、內部稽核、管理審查）

---

## ISO 22000 的四大系統要素

| 要素 | 說明 |
|---|---|
| **互動式溝通（Interactive Communication）** | 在整個供應鏈中，上下游業者之間關於食安資訊的系統性溝通機制 |
| **系統管理（System Management）** | 以組織架構、角色責任、文件管理系統支撐食安目標 |
| **前提方案（PRPs, Prerequisite Programs）** | 相當於 GHP 和基本衛生規範的系統化文件版本 |
| **HACCP 原則** | 危害分析和關鍵管制點（和 HACCP 標準一致）|

---

## ISO 22000 vs HACCP vs GHP 的三層關係

```
GHP（衛生基礎）→ HACCP（危害管控）→ ISO 22000（系統化整合）
```

**簡單比喻**：
- GHP：確保廚房是乾淨的
- HACCP：找出哪個步驟最可能出問題，並建立管控
- ISO 22000：把上面兩個整合進公司的完整管理架構，並加上供應鏈管理和持續改善機制

## 台灣長照食品業使用 ISO 22000 的情境

**[✅ Fact-Checked]** ISO 22000 認證在台灣食品業的應用：
- 漢典食品等出口導向業者，持有 ISO 22000 認證作為進入歐盟、英國市場的基本合規文件
- 大型食品集團（統一、味王等的子品牌）通常持有 ISO 22000
- 中小型長照業者：目前以 HACCP + GHP 合格為主，ISO 22000 是升級路徑

**ISO 22000 的商業意義**：
1. **出口市場必備**：歐洲、美國、日本的大型食品採購商，通常要求供應商持有 ISO 22000 或 FSSC 22000（ISO 22000 的加強版）
2. **B2B 大客戶加分**：部分台灣大型醫療集團（臺大醫院、長庚等）的採購評選中，ISO 22000 是加分項
3. **企業管理升級**：取得 ISO 22000 的過程本身，迫使業者系統化整理所有的食安和品質流程，對業者內部管理有明顯提升效果

---

## FSSC 22000：進一步加強版

**FSSC 22000**（Food Safety System Certification 22000）是在 ISO 22000 基礎上，由荷蘭食品安全認證基金會（FSSC）加入了 ISO/TS 22002 的補充要求，是全球食品業最指標性的認證（Coca-Cola、雀巢等跨國食品企業要求供應商持有 FSSC 22000）。
