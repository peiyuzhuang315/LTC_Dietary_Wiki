# 詞條：ISO 22000 認證（小白友善版）
> **所屬面向**：認證/標章面向 | **建立日期**：2026-03-18 | **事實查核**：已完成

## 🔷 一分鐘秒懂

如果 [HACCP認證](HACCP認證.md) 是「讓台灣機構相信你的食品安全管控」，那麼 **ISO 22000** 就是「讓日本買家、歐洲通路商、國際醫療集團相信你」的護照。

ISO 22000 是由國際標準化組織（ISO）頒布的食品安全管理系統認證，整合了 HACCP 的危害管控邏輯 + ISO 9001 的流程改善文化。取得 ISO 22000，代表你的食品安全管理已達「國際企業級水準」。

---

## 🔷 核心知識白話轉譯

**三明治解說法**：
> 把 ISO 22000 想成「食品版的 ISO 9001」——不只要求「廚房乾不乾淨」（那是 GHP 的事），而是要求「整個公司從原料採購、製程管控到產品出貨的每一個環節，都有系統化的風險預防與持續改善機制」。取得 **[ISO 22000認證](ISO 22000認證.md)** 的業者，等於向買家宣示：「我們的制度不靠個別員工的良心，而是靠系統保障品質。」

**ISO 22000 vs HACCP 的差異**：

| 比較點 | HACCP | ISO 22000 |
|---|---|---|
| 範疇 | 以生產線危害為核心 | 整個食品安全管理系統 |
| 國際認可度 | 台灣主要採購接受 | **全球通用** |
| 文件要求 | CCP記錄為主 | 管理手冊+程序書+記錄三層 |
| 審核方式 | 政府或第三方評鑑 | **國際認可第三方機構** |

## 🔷 硬核事實與費用

**[✅ Fact-Checked]** 台灣 ISO 22000 認證費用與時程（參考值）：
- 現場稽核費：約 **15,000元/人天**
- 證書費：**5,000元/件**
- 年費：**5,000元/年**
- 顧問輔導費（50人以下公司）：另計，整體可能需 **10-30萬元**
- 認證時程（有HACCP基礎者）：**2-4個月**；從頭建置需更長
- 有效期：**3年**，期滿需複審
> 來源：[isolution.com.tw](https://isolution.com.tw)、[cgprdi.org.tw](https://cgprdi.org.tw)

**[✅ Fact-Checked]** 漢典食品已通過 **ISO 22000 + HACCP 雙認證**，是台灣長照膳食 B2B 市場取得此組合認證的代表性業者。
> 來源：[frozenfoods.com.tw](https://frozenfoods.com.tw)

> **申請管道**：可透過 CGP 顧問技術服務公司（cgprdi.org.tw）、TQCSI-Taiwan（tqcsi-taiwan.com.tw）等國際認證機構申請。

## 🔷 落地應用與避坑指南

**何時考慮申請 ISO 22000？**

| 情境 | 建議 |
|---|---|
| Day 1-180（剛起步）| 先完成 GHP → 推進 HACCP，ISO 22000 不是第一優先 |
| 已有 HACCP，要拓展日本/歐洲市場 | **立即申請 ISO 22000** |
| 要爭取中大型醫療集團或連鎖機構採購 | ISO 22000 是採購規格要求 |

**最大的坑**：業者常低估 ISO 22000 對「管理層投入程度」的要求——不是只有品保部門的事，高層主管必須簽署食品安全政策聲明、定期參加管理審查會議，否則在年度審核時會被要求改善。
