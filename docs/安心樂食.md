# 詞條：安心樂食（Comfortable & Safe Eating）
> **所屬面向**：經營/營運面向 | **改寫日期**：2026-03-18

---

「安心樂食」這四個字，是台灣長照膳食產業過去幾年最有力量的政策口號。它的背後邏輯簡單卻深刻——**吃東西這件事，應該讓長者既安全（不嗆咳、不噎食）又快樂（有滋味、有尊嚴）**。安全和快樂，在過去的長照膳食設計裡經常被當成互相拉扯的矛盾：食物做得越安全，往往就越無味、越難看；反過來讓食物好看好吃，又不一定夠安全。

**[安心樂食](安心樂食.md)** 的核心主張，就是主張這兩者應該同時達到，而不是非此即彼的取捨。

### 「安心」和「樂食」各代表什麼？

業者在落地時，應該把這四個字拆成兩條可執行的產品設計方向：

| 維度 | 目標 | 執行工具 |
|---|---|---|
| **安心**（食安與安全）| 吞嚥安全、食材衛生、法規合規 | IDDSI 質地分級、留存檢體、HACCP 認證 |
| **安心**（照護安全）| 住民個別化飲食需求被滿足 | SLP 處方 + MDT 會議 |
| **樂食**（官感體驗）| 食物有味道、有外觀、有選擇感 | 味覺設計、模具塑形、菜單選擇權 |
| **樂食**（社會連結）| 吃飯不只填肚子，是每日最重要的社交時刻 | 共餐環境設計、集體用餐儀式感 |

### 這個概念怎麼變成業者的銷售主張？

把「安心樂食」轉化為業者的商業語言，有三個層次：

**對院長說**：「我們的膳食服務，讓您的住民在用餐時既不嗆咳（安心），又感到被尊重（樂食）——這兩件事加在一起，直接影響住民家屬的口碑評價。」

**對護理長說**：「我們的每一份食物都有 IDDSI 等級書面記錄，出事了您有文件保護自己；而且我們的食物看起來不像醫院伙食，住民比較願意吃，進食量達標率更高。」

**對評鑑委員說**：「我們的膳食服務以『安心樂食』為設計核心，整合了質地安全管控（IDDSI 分級）、尊嚴進食設計（食物外觀保留）和多職種協力會議（MDT）三個面向。」

**[⚠️ 待查證]** 「安心樂食」目前並非衛福部官方計畫的正式用語，各機構和業者使用方式略有差異。建議追蹤農業部、衛福部 2026 年是否有整合性政策公告將此概念納入正式框架。
