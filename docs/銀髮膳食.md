# 詞條：銀髮膳食（Senior Diet / Silver Meal）
> **所屬面向**：使用者/照護需求面向 | **改寫日期**：2026-03-18

---

「銀髮膳食」是對「為老年人設計的飲食」最廣義的統稱，它的範疇涵蓋從健康長者的「預防性高蛋白飲食」，一路延伸到重度失能者需要的「IDDSI 4 級均質食品」——是一個橫跨整個老年照護光譜的品類概念。

**[銀髮膳食](銀髮膳食.md)** 這個詞在台灣商業場合使用最廣泛——業者向院長推銷時說的是「銀髮膳食解決方案」；Eatender 認證的文件上寫的是「銀髮友善食品」；臨床上 SLP 開立的是「IDDSI 飲食等級」。三個詞彙說的都是同一件事的不同面向。

### 銀髮膳食設計的黃金三角

任何一份優秀的銀髮膳食，都必須同時滿足三個核心要素：

```
         ✅ 安全
        /         \
       /           \
  ✅ 美味 ——— ✅ 營養足夠
```

| 要素 | 失敗案例 | 成功設計 |
|---|---|---|
| **安全（不嗆咳）** | 質地太硬/太稀，超過住民 IDDSI 安全等級 | SLP 處方 + 業者 IDDSI 認證食品 |
| **美味（願意吃）** | 均質打泥後一盤棕色糊狀，零食慾 | TFL 技術保留色香，模具塑形 |
| **營養足夠（夠量）** | 食量小但攝取量不足（蛋白質缺口）| 高密度配方 + 補充品搭配 |

**[✅ Fact-Checked]** 台灣長者實際蛋白質攝取量低於 ESPEN 建議量的比例：約 **50% 以上**的社區居住長者，日常飲食的蛋白質攝取量未達 1.0 g/kg/天的最低建議量。
> 來源：台北市社區營養調查（多篇學術研究）

### 銀髮膳食的在台灣的市場邊界

業者要清楚「銀髮膳食」這個詞的使用邊界：
- **B2B 語境（對院長說）**：不要只說銀髮膳食，要說「符合 Eatender / IDDSI 的銀髮膳食」——加認證才有說服力
- **B2C 語境（對家屬說）**：可以說「專為長者設計的銀髮膳食」但要避免「老人食品」的負面標籤
- **廣告語境**：非健康食品認證產品，不得聲稱「改善肌少症」等醫療效果
