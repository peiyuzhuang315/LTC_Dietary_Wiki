# 肌少症（Sarcopenia）

> **分類**：老年醫學 ｜ 照護需求 ｜ 營養介入  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：家屬、照服員、長照業者、醫護人員

---

## 什麼是肌少症？

**肌少症**（英文：Sarcopenia，源自希臘文 sarx「肉/肌肉」+ penia「貧乏/流失」）是一種以**骨骼肌肉量減少、肌肉力量下降、身體功能退化**為核心特徵的老年症候群。它是一種漸進性的生理變化，早期症狀不明顯，往往等到長者摔倒、走路越來越慢、無法自行從椅子站起來，家屬才意識到問題的嚴重性。

肌少症在 2016 年已被納入國際疾病分類（ICD-10-CM，代碼 M62.84），正式確認為一項需要積極診斷和治療的疾病，而非「老了自然的事」。

## 肌肉是怎麼流失的？——生理機制解析

人類的骨骼肌肉量在 25-30 歲達到高峰，此後開始緩慢下降。**40 歲後每十年流失約 8%**，70 歲後速率加快至**每十年流失 15%**。這種流失背後有幾個相互交織的機制：

### 1. 蛋白質合成與分解的失衡
年輕人的肌肉每天都在進行「蛋白質合成」和「蛋白質分解」的動態平衡，老化使得：
- **合成效率下降**：同樣份量的蛋白質飲食，老年人觸發肌肉蛋白質合成的能力（即「合成代謝反應 Anabolic Response」）只有年輕人的 70-80%
- **分解速率不變或上升**：發炎因子（如 IL-6、TNF-α）隨老化增加，持續促進肌肉分解

### 2. 荷爾蒙環境改變
- **睾固酮（雄性激素）**：男性 50 歲後逐年下降，肌肉合成激素減少
- **生長激素（IGF-1）**：分泌減少，肌肉修復和再生能力下降
- **雌激素**：女性停經後保護肌肉的雌激素驟降，是女性肌少症進展快於男性的主因之一

### 3. 神經肌肉退化
第二型肌肉纖維（快縮肌，負責爆發力）在老化過程中比第一型（慢縮肌，耐力型）流失更快，讓老年人在「突然要抓東西防止跌倒」的瞬間動作中特別脆弱。

### 4. 慢性發炎（Inflammaging）
許多長照住民同時患有糖尿病、高血壓、慢性腎臟病等慢性病，這些都會提高身體的慢性低度發炎狀態，持續消耗肌肉組織。

## 肌少症有多嚴重？台灣現況

**[✅ Fact-Checked]** 台灣肌少症盛行率數據：
- 台灣 **65 歲以上社區長者**，肌少症盛行率約 **34.1%**（國健署 2017-2020 年調查）
- 換算人口：約 **160 萬名**台灣老年人達肌少症診斷標準
- **長照機構住民**的肌少症比例更高，部分研究顯示超過 **50%**
> 來源：[ankecare.com](https://ankecare.com)、國民健康署

**肌少症的連鎖健康後果**：

| 直接影響 | 後續風險 |
|---|---|
| 行動能力下降 | 跌倒風險增加 2-3 倍 |
| 吞嚥肌群退化 | 吸入性肺炎風險上升 |
| 免疫力下降 | 術後恢復期延長、感染風險增加 |
| 代謝功能退化 | 血糖控制更困難（糖尿病惡化）|
| 骨密度同步流失 | 骨折風險大幅提升 |

## 如何判斷家人是否有肌少症？

### 家屬可以做的簡單觀察

| 觀察項目 | 警訊 |
|---|---|
| **步行速度** | 在平地以正常速度行走 6 公尺，超過 **10 秒**完成（即步速 < 0.6 m/s）|
| **握力測試** | 男性 < **26 公斤**、女性 < **18 公斤**（一般門診握力計測量）|
| **從椅子站起** | 雙手交叉放胸前，**5 次起立**超過 12 秒 |
| **小腿圍** | 男性 < **34 cm**、女性 < **33 cm**（用軟尺測量小腿最粗處）|

> **注意**：以上為簡易篩查，若出現警訊，應前往老年醫學科或復健科進行正式評估。

### 臨床診斷標準（AWGS 2019，亞洲肌少症工作組）

**[✅ Fact-Checked]** 台灣臨床採用的亞洲肌少症診斷標準（AWGS 2019）：
1. **肌肉量測定**：DXA（雙能量X光吸光式測定儀）或 BIA（生物阻抗分析）測量四肢肌肉量
   - 男性 < **7.0 kg/m²**；女性 < **5.4 kg/m²**（以身高平方校正）
2. **肌力測定**：握力測試（男 < 28 kg；女 < 18 kg）
3. **功能測定**：6公尺步速 < 1.0 m/s 或 5次起立 > 12秒
> 來源：亞洲肌少症工作組（Asian Working Group for Sarcopenia, AWGS）2019 年共識報告

## 飲食是核心介入工具

### 蛋白質：最關鍵的飲食營養素

**[✅ Fact-Checked]** ESPEN（歐洲臨床營養學會）2019 年老年人蛋白質建議量：
- 健康老年人：**1.0-1.2 g/公斤體重/天**（即 60 公斤長者每日需 60-72 克蛋白質）
- 有肌少症風險者：**1.2-1.5 g/公斤體重/天**
- 疾病恢復期老年人：**≥ 1.5 g/公斤體重/天**

**現實問題**：60 公斤的長照住民每日至少需要 72 克蛋白質，相當於：
- 雞胸肉（每 100g 含 31g 蛋白質）：需吃 **232 克**
- 雞蛋（每顆 7g 蛋白質）：需吃 **10 顆**

對食量只剩正常人 60% 的長者而言，光靠三餐幾乎無法達標——這就是[蛋白質補充](蛋白質補充.md)品存在的核心理由。

### 蛋白質補充品的選擇原則

| 補充品類型 | 優點 | 缺點 |
|---|---|---|
| **乳清蛋白（Whey Protein）** | 消化吸收最快，富含支鏈胺基酸（BCAA）| 乳糖不耐者需注意 |
| **黃豆分離蛋白** | 植物性，低脂 | 合成效率略低於乳清蛋白 |
| **均衡配方（如護力養）** | 含蛋白質+ 脂肪 + 碳水化合物，完整熱量與營養 | 單一蛋白質濃度較低 |
| **膠原蛋白** | 市場行銷吸引力強 | **必需胺基酸不完整，對肌少症防治幫助有限** |

### 補充蛋白質的最佳時機

研究顯示，運動後 30-60 分鐘內補充蛋白質（特別是乳清蛋白）可以最大化肌肉蛋白質合成。長照現場可行的做法：早餐後的口腔運動或站立訓練，搭配含有蛋白質的點心。

### 其他關鍵營養素

- **維生素 D**：促進肌肉蛋白質合成，台灣長者維生素 D 不足比例超過 50%
- **Omega-3 脂肪酸**：EPA/DHA 可減少肌肉炎症，改善肌肉合成代謝阻抗
- **肌酸（Creatine）**：輔助肌力訓練效果，長照場域較少應用，但有研究支持

## 運動：飲食介入不可缺少的搭檔

飲食介入必須搭配**阻力訓練（Resistance Training）**才能發揮最大效益。對長照住民，阻力訓練不一定需要啞鈴：

- **彈力帶訓練**：坐姿即可進行，小腿上抬、手臂推拉
- **功能性訓練**：站起、坐下、踏步
- **水中運動**：對關節友善，有游泳池的機構適合
- SLP、OT、PT 三種治療師在 MDT 框架下共同設計個人化訓練計畫

## 長照膳食業者的應用角度

肌少症是長照膳食業者**最有說服力的銷售切入點**之一，因為它：
1. 有**清楚的數字指標**（MNA-SF 評分、體重、握力）可以追蹤改善效果
2. 有**明確的飲食介入空間**（蛋白質補充量化目標）
3. 家屬**容易理解**（「肌肉流失→跌倒→骨折→死亡」的直觀邏輯）

**建議業者的行動方案**：
1. 與合作機構的護理長建立「每季 MNA-SF 篩查 + 高風險住民蛋白質補充方案」的閉環服務
2. 提供菜單的「每份蛋白質含量標示」，讓 RD 和護理長一目了然
3. 合作 SLP，確認吞嚥障礙住民可以安全食用業者提供的補充品（IDDSI 等級確認）
