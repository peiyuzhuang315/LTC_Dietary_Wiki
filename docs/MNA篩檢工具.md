# MNA 篩檢工具（Mini Nutritional Assessment）

> **分類**：評估工具 ｜ 市場分析 ｜ 專業人員工具  
> **更新日期**：2026-03-18 ｜ **適合閱讀對象**：護理師、照服員、長照業者

## 什麼是 MNA？

**MNA**（Mini Nutritional Assessment，簡式老人營養評估量表）是目前全球長照和老年醫學場域**最廣泛使用**的老年人營養風險篩查工具，由瑞士雀巢研究中心（Nestlé Research Center）和歐美多所大學共同設計，1994 年首次發表，已累計在全球超過 300 篇臨床研究中得到驗證。

MNA 的核心目標是：用**簡單、快速（5-10 分鐘）的問卷和測量**，辨識出營養不足（Malnutrition）或營養不足風險（At Risk）的老年個案，讓照護團隊可以在情況惡化前提早介入。

---

## MNA 的兩個版本

### MNA-SF（簡短版，Short Form，6 題）

MNA-SF 是日常篩查中最常用的版本，用 **6 個問題完成初步篩查**，總分 14 分：

| 問題 | 分項 | 分數範圍 |
|---|---|---|
| **A. 食慾下降**：最近 3 個月內，是否因食慾不佳、消化問題、咀嚼或吞嚥困難而減少進食？ | 嚴重減少/中度減少/未減少 | 0-2 |
| **B. 體重下降**：最近 3 個月內，體重是否減輕？ | > 3kg / 不知道 / 1-3kg / 未減輕 | 0-3 |
| **C. 活動能力**：活動能力如何？ | 臥床/輪椅受限 / 可下床但不能出門 / 可出門 | 0-2 |
| **D. 心理壓力或急性疾病**：過去 3 個月是否有心理壓力或急性疾病？ | 是 / 否 | 0-2 |
| **E. 神經精神問題**：是否有失智症或憂鬱症？ | 嚴重 / 輕度 / 無 | 0-2 |
| **F. BMI 或小腿圍** | BMI < 19: 0分 / 19-21: 1分 / 21-23: 2分 / ≥23: 3分 | 0-3 |

**判讀標準**：
- **12-14 分**：營養狀態正常（Normal Nutritional Status）
- **8-11 分**：有營養不足風險（At Risk of Malnutrition）
- **0-7 分**：營養不足（Malnourished）

### MNA 完整版（Full Form，18 題）

MNA-SF 初篩發現「有風險」或「營養不足」後，完整版提供更細緻的評估，包含飲食多樣性、每日液體攝取量、自我評估健康狀況等。

## 台灣長照的 MNA 應用現況

**[✅ Fact-Checked]** 台灣長期照護相關規範：
- 長照機構住民依法必須定期進行**營養評估**，MNA 是最常被引用的標準工具
- 衛福部的長照機構評鑑標準中，**住民的營養評估記錄完整性**是重要的查核項目
- 部分縣市要求機構每 **3-6 個月**進行一次 MNA 篩查，高風險住民更需要縮短間隔
> 來源：衛福部長照評鑑基準

**台灣研究發現**：台灣長照機構住民中，**MNA 篩查結果顯示有營養不足或風險者**，比例通常在 **40-60%** 之間——這是一個驚人的高比例，代表超過半數的機構住民存在需要積極介入的營養問題。

## MNA 對業者的商業意義

### 用 MNA 數據說話

MNA 分數可以成為業者服務效果最有力的量化佐證：

**服務前**：A 機構的 50 位住民，MNA-SF ≤ 11 分（有風險）的比例是 68%

**業者介入後 6 個月**：每餐增加蛋白質補充品 + 升級菜單設計，MNA-SF ≤ 11 分的比例降為 42%

**對院長說**：「我們的服務讓貴機構高風險住民比例降低了 26 個百分點。這不只是統計數字，而是 13 位住民的骨折風險、感染風險、住院率都同時降低。」

### 業者可以協助機構執行的 MNA 服務

1. **提供 MNA-SF 問卷和評分表**（免費，雀巢官網可下載繁中版）給機構護理師使用
2. **協助解讀 MNA 結果**：依分數提供對應的飲食建議（不需要 RD 常駐，業者也能提供初步指引）
3. **建立季度追蹤報告**：每季彙整機構住民的 MNA 分數變化，提交給院長——這是業者「成效可見化」的核心工具

## MNA 的限制

- MNA-SF 的「體重下降」和「食慾」是自我陳述（住民若失智，準確性下降）
- 在急性疾病期間或術後，MNA 結果往往暫時偏低，不代表長期營養狀態
- MNA 是**篩查工具**，不是完整的營養評估——篩查陽性後，需要 RD 進行完整的個別化評估
