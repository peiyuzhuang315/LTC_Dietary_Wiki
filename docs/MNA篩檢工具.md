# 詞條：MNA 篩檢工具（Mini Nutritional Assessment）
> **所屬面向**：市場分析面向 | **改寫日期**：2026-03-18

---

如果要在長照場域「讓院長看到你的服務有成效」，你需要的不是一份好吃報告，而是一個可追蹤的數字。**[MNA篩檢工具](MNA篩檢工具.md)**（Mini Nutritional Assessment，迷你營養評估量表）就是這個數字最方便的來源。

MNA 是目前全球最廣泛使用的老年人營養狀態篩檢工具，由瑞士雀巢研究中心與多所大學共同開發，短版（**MNA-SF，6 題**）只需 3-5 分鐘完成，無需抽血，任何護理人員都能操作。

### MNA-SF 六題快速解析

| 題號 | 評估項目 | 評分範圍 |
|---|---|---|
| Q1 | 過去三個月是否因食慾下降、消化問題等減少飲食量 | 0-2 |
| Q2 | 過去三個月是否有體重減輕 | 0-3 |
| Q3 | 移動能力 | 0-2 |
| Q4 | 過去三個月是否有心理壓力或急性疾病 | 0-2 |
| Q5 | 神經心理問題（失智或憂鬱）| 0-2 |
| Q6 | BMI 或小腿圍（Calf Circumference）| 0-3 |
| **總分** | **最高 14 分** | — |

**判讀標準**：
- **≥ 12 分**：營養狀態正常，繼續一般膳食
- **8-11 分**：有營養不良風險，需主動膳食介入（蛋白質補充、MDT 會議）
- **≤ 7 分**：營養不良，需立即完整評估及強化介入計畫

**[✅ Fact-Checked]** MNA-SF 在多個台灣長照機構研究中被驗證其效度，台灣老年醫學會推薦長照機構每個入住時評估一次、此後每季追蹤。
> 來源：[ankecare.com](https://ankecare.com)

### 業者怎麼把 MNA 變成合約工具？

每季帶著護理師執行住民 MNA-SF 篩查，把結果整理成「**高風險住民名單 → 強化蛋白質干預計畫 → 下一季追踪再評估**」，形成閉環報告。這份季報就是你在合約續簽時最有力的成效憑證。
