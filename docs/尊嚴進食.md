# 詞條：尊嚴進食（Dignified Eating）
> **所屬面向**：使用者/照護需求面向 | **改寫日期**：2026-03-18

---

「讓奶奶自己拿湯匙吃飯」——這句話背後藏著無數照護現場的情感張力。因為當奶奶的手開始抖、食物一口口掉在圍兜上、照護員必須從側邊協助每一口的時候，「吃飯」這件原本最平凡的生活日常，就悄悄變成了尊嚴的戰場。

**[尊嚴進食](尊嚴進食.md)** 是一個以「人」為核心的照護哲學——它主張：即使一個人的身體功能退化了，他對「如何從事最基本的生命活動（吃飯）」仍保有選擇的權利和被尊重的需求。在長照膳食業，把這個哲學轉化成產品和服務設計，是差異化競爭的最高層次。

### 尊嚴進食的四個實踐維度

| 維度 | 失去尊嚴的樣子 | 尊嚴的設計 |
|---|---|---|
| **自主性** | 照護員用湯匙強餵 | 適合殘存功能的輔具（防傾碗、彎柄匙）讓住民自行進食 |
| **外觀** | 均質後的棕色糊狀讓人沒食慾 | 模具塑形（食物外觀保留）、擺盤、色彩對比 |
| **社會性** | 在床邊一個人進食 | 集體用餐空間、共餐氣氛設計 |
| **選擇感** | 每天同一種菜單沒有選擇 | 每週提前勾選菜色（在質地安全範圍內）|

### 「外觀」是被嚴重低估的尊嚴要素

研究顯示，老年人的進食量有很大比例受到「食物視覺呈現」的影響。一份「看起來像正常食物但已均質化至 IDDSI 4 級」的魚排，遠比同等營養的「棕色糊狀泥醬」能誘發更高的食慾和進食量——而進食量直接關聯到蛋白質達標率，進而影響肌少症防治和住院率。

這就是 3D 列印食物（如德國 Biozoon 的 SMOOTHFOOD 系列）在概念上令人興奮的原因：它讓 IDDSI 4 級的泥狀食品可以呈現成「雞腿形」或「蔬菜形」，完整還原食物的社會意義。而台灣的格林全食物 TFL 技術（「不去皮、不去渣」），在維持食材顏色和天然香氣上，也是往同一個方向努力的嘗試。
