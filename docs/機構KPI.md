# 詞條：機構 KPI（Institutional Key Performance Indicators）
> **所屬面向**：市場分析面向 | **改寫日期**：2026-03-18

---

「你的服務讓機構進步了什麼？」——這是合約到期前院長內心的那個問題。業者如果無法回答，就只能跟著院長的殺價進退。**[機構KPI](機構KPI.md)** 是業者把「服務有成效」這件事量化的工具，也是從「cost center（成本中心）」轉型為「value creator（價值創造者）」的關鍵語言。

長照膳食相關的機構 KPI，可以分成三個維度：

### 三維度 KPI 追蹤架構

| KPI 維度 | 具體指標 | 追蹤頻率 | 業者貢獻角色 |
|---|---|---|---|
| **安全性**| 吸入性肺炎發生率次數/季 | 每季 | 食物 IDDSI 等級正確性 |
| **安全性**| 噎食事件次數/季 | 每季 | 質地合規、增稠劑使用 |
| **安全性**| 食安事件（食物中毒等）| 即時 | 留存檢體、GHP/HACCP |
| **營養狀態**| MNA-SF 風險比例（≤11分住民%）| 每季 | 蛋白質補充、高密度菜單 |
| **營養狀態**| 體重維持率（非刻意減重 < 5%/季）| 每月 | 進食量記錄、配方調整 |
| **住民滿意度**| 進食滿意度問卷分數（1-5 分）| 每半年 | 菜色選擇、口感設計 |
| **評鑑合規**| 膳食相關評鑑項目得分 | 每次評鑑 | 文件合規、SOP 管理 |

### 業者的 KPI 季報格式

建議業者提供給機構的季報範本包含以下三個區塊：

```
【長照膳食 KPI 季報 — 2026 Q1】
機構名稱：XXX 養護機構
合作業者：XXX食品

■ 安全指標
  - 吸入性肺炎事件：0 次（上季：1 次）⬇️ 改善 100%
  - IDDSI 等級符合率：98%（新進住民 2 名正在評估中）

■ 營養狀態
  - MNA-SF ≤11 分住民：32人中 8 人（25%）
  - 已啟動強化蛋白補充方案：8 人（100%）
  - 體重維持率：94.7%（監控中）

■ 住民滿意度（問卷 32 份，回收 28 份）
  - 整體滿意度：4.2 / 5.0
  - 口感滿意度：4.0 / 5.0

■ 下季行動計畫
  - 新增 3 道 IDDSI 4 級甜點選項
  - SLP 訪視：4 月 15 日排定
```
