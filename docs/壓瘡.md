# 壓瘡

## 什麼是壓瘡？

壓瘡，又稱褥瘡或壓力性損傷，是指皮膚及／或下層組織，因長時間受到壓力、剪力（摩擦力）或兩者合併作用，導致局部血液循環不良、組織缺氧壞死而形成的損傷。這類損傷通常發生在骨頭突出處，例如薦骨（尾椎）、髖部、腳踝、腳跟、手肘、枕骨（後腦勺）等部位。壓瘡的嚴重程度可分為不同期別，從皮膚完整但出現紅斑（第一期），到皮膚全層損傷、可見骨骼、肌腱或肌肉（第四期），甚至可能因組織壞死過深而無法判斷嚴重程度（無法分期），或是深層組織損傷但皮膚完整（疑似深層組織損傷）。

壓瘡的形成不僅與壓力大小和時間有關，還受到多重內外在因素影響。內在因素包括濕氣（如大小便失禁）、摩擦力、剪力、營養狀況不良、活動能力受限、感覺功能受損（如糖尿病神經病變）、慢性疾病（如糖尿病、周邊血管疾病）及年齡增長導致皮膚變薄脆弱等。外在因素則包含不適當的床墊或坐墊、不正確的翻身或移動方式等。對於長期臥床、行動不便、意識不清或感覺功能受損的長輩而言，壓瘡是常見且嚴重的併發症，會嚴重影響生活品質，導致劇烈疼痛、活動受限、食慾不振，甚至可能因傷口感染而引發蜂窩性組織炎、骨髓炎、敗血症等危及生命的狀況，因此預防與早期介入顯得至關重要。

---

## 為什麼重要

壓瘡對於長期照護的家屬或機構來說，是一個不容忽視的重大議題，其重要性體現在多個層面：

首先，**預防勝於治療**。預防壓瘡的發生，遠比治療壓瘡來得容易且成本效益更高。一旦壓瘡形成，不僅會帶給長輩極大的疼痛與不適，影響其食慾、睡眠與整體身心健康，更可能因傷口感染而延長住院時間，甚至需要進行清創手術或植皮，大幅增加醫療費用與照護負擔。

其次，**影響生活品質與尊嚴**。壓瘡的疼痛和不適會嚴重降低長輩的生活品質，使其感到焦慮、沮喪，甚至影響社交活動。傷口換藥的頻繁與不便，也可能讓長輩感到失去尊嚴。良好的壓瘡預防與照護，能有效提升長輩的舒適度與生活品質。

再者，**加重家庭照護負擔**。對於家屬而言，照顧一位有壓瘡的長輩，意味著需要投入更多的時間、精力與金錢在傷口換藥、定期翻身、營養補充等方面，長期下來容易產生身心俱疲的壓力，甚至影響家庭關係。

最後，**關乎機構照護品質與聲譽**。對於長照機構而言，壓瘡的發生率與嚴重程度，是評估照護品質的重要指標之一。高壓瘡發生率不僅影響機構的聲譽，可能導致家屬信任度降低，也可能面臨相關的法律責任與賠償風險。因此，建立完善的壓瘡預防與照護計畫，不僅是提升長輩生活品質的關鍵，更是展現專業照護能力與倫理責任的重要環節，有助於建立機構的專業形象與社會信任。

## 實際操作指南

### 壓瘡風險評估與預防

預防壓瘡是照護工作的核心，透過以下策略可有效降低壓瘡發生率，並應將這些措施融入日常照護流程中：

1.  **定期評估風險**：
    *   **標準化工具應用**：使用國際認可的標準化評估工具，例如「布雷登量表 (Braden Scale)」，評估長輩的感覺知覺、濕氣、活動力、移動力、營養及摩擦力/剪力等六個面向。量表分數越低，表示壓瘡風險越高。
    *   **動態評估**：壓瘡風險並非一成不變，應根據長輩的健康狀況變化（如疾病惡化、營養攝取改變、活動力下降等）定期重新評估，並在入院、轉床、病情變化時立即進行評估。
    *   **制定個別化計畫**：根據評估結果，制定並執行個別化的壓瘡預防計畫，例如高風險者可能需要更頻繁的翻身、更完善的減壓輔具及更密集的皮膚檢查。

2.  **定時翻身與正確擺位**：
    *   **翻身原則**：對於長期臥床的長輩，應至少每2小時翻身一次，並記錄翻身時間與姿勢。坐輪椅者則應每15-30分鐘協助或指導其改變重心，或抬高臀部10秒鐘，以重新分配壓力。
    *   **擺位技巧**：避免直接壓迫骨突處，可使用枕頭、海綿墊或減壓墊等輔具，將身體重量分散，避免兩骨突處直接接觸。例如，側臥時可將枕頭置於兩膝之間，避免膝蓋及腳踝互相壓迫；仰臥時可將枕頭置於小腿下方，使足跟懸空。
    *   **床頭抬高限制**：床頭抬高角度不宜超過30度，以減少剪力對薦骨部位的影響，特別是在進食或進行其他活動後，應盡快將床頭角度調低。

3.  **維持皮膚完整與清潔**：
    *   **每日檢查**：每天至少檢查一次全身皮膚，特別是骨突處（如薦骨、髖部、腳跟、手肘、枕骨），觀察是否有發紅、變色、水泡、破皮或皮膚溫度異常（局部發熱或冰冷）等初期徵兆。對於深色皮膚的長輩，應特別注意皮膚顏色變化、水腫、硬度或觸痛感。
