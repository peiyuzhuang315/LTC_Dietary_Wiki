# 氣墊床補助

氣墊床補助，是指政府為減輕失能者家庭的經濟負擔，針對符合特定條件的失能者，提供購買或租賃「氣墊床」這類醫療輔具的部分費用補助。氣墊床是一種透過交替充氣與洩氣來分散身體壓力的特殊床墊，主要目的是預防或改善因長期臥床而產生的壓瘡（褥瘡），是長期臥床者居家照護中不可或缺的重要輔具之一。這項補助屬於政府長期照顧服務中的「輔具服務及居家無障礙環境改善服務」項目，旨在協助失能者維持生活品質，並減輕照顧者的負擔。

## 為什麼重要

氣墊床補助對於長期照顧家庭或機構而言，具有多重且深遠的意義：

1.  **預防與改善壓瘡，提升生活品質：** 長期臥床者最常見且嚴重的併發症之一就是壓瘡。壓瘡不僅會造成劇烈疼痛，影響患者的舒適度與尊嚴，更可能引發感染，甚至危及生命。氣墊床透過動態減壓原理，能有效分散身體壓力，大幅降低壓瘡發生的風險，對於已產生壓瘡者，也能提供良好的復原環境。這直接關係到失能者的生活品質與健康狀況。
2.  **減輕家庭經濟負擔：** 一張功能良好的氣墊床價格不菲，對於許多長期照顧家庭而言，是一筆沉重的開銷。政府提供的補助能有效減輕這筆費用，讓家庭在面對長期照護的龐大支出時，能獲得實質的經濟支援，不必為了省錢而犧牲被照顧者的健康與舒適。
3.  **降低照顧者壓力：** 照顧一位長期臥床的家人，本身就是一項艱鉅的任務。若被照顧者又因壓瘡而產生疼痛、感染，照顧者除了要處理日常照護，還需花費更多心力處理傷口照護、就醫等問題，身心壓力倍增。氣墊床的運用能有效減少壓瘡的發生，間接降低照顧者的照護負擔與心理壓力。
4.  **促進居家照護可行性：** 許多家庭希望能在熟悉的居家環境中照顧失能家人，但缺乏適當的輔具往往是阻礙。氣墊床補助讓家庭更容易取得必要的輔具，使居家照護更具可行性與品質，避免因輔具不足而被迫選擇機構照護。
5.  **體現社會福利與人道關懷：** 這項補助政策彰顯了政府對弱勢群體的關懷，確保失能者能獲得應有的照護與尊嚴，是健全社會福利體系的重要一環。

## 實際操作指南

申請氣墊床補助的流程，主要可分為以下幾個步驟：

1.  **確認申請資格：**
    *   **失能等級：** 申請者需經長期照顧管理中心（簡稱照管中心）評估，符合長期照顧服務對象（如失能等級2級以上）。
    *   **居住地：** 需設籍並實際居住於台灣。
    *   **輔具需求：** 經專業評估確認有氣墊床的使用需求。

2.  **提出申請：**
    *   **聯繫照管中心：** 撥打全國統一專線 **1966**，或親洽各縣市長期照顧管理中心，說明您的需求。
    *   **安排評估：** 照管中心會派員到府進行初步評估，確認失能等級及長照服務需求。
    *   **輔具評估：** 針對氣墊床這類輔具，通常會由專業治療師（物理治療師或職能治療師）進行專業評估，確認個案的身體狀況、臥床程度、壓瘡風險等，並建議適合的氣墊床類型。這份評估報告是申請補助的重要依據。

3.  **核定與購買：**
    *   **審核：** 照管中心會依據您的申請資料、失能評估結果及輔具評估報告進行審核。
    *   **核定公文：** 審核通過後，您會收到一份「輔具服務核定函」，上面會載明核定的輔具項目、補助金額上限及有效期限。**請務必在收到核定函後再進行購買或租賃。**
    *   **購買或租賃：** 憑核定函至合法立案的輔具廠商選購或租賃符合核定規格的氣墊床。請務必索取統一發票或收據，上面需載明品項、數量、金額及購買者姓名。

4.  **核銷撥款：**
    *   **備齊文件：** 準備核定函、購買發票或收據正本、申請人（或代理人）存摺影本、輔具評估報告書等文件。
    *   **提出核銷：** 將上述文件送交至原申請的照管中心或指定單位。
    *   **撥款：** 經審核無誤後，補助款項會撥入申請人指定的帳戶。
