# 氣墊床補助

您好，當家中長輩或親人因疾病或意外而需長期臥床時，如何預防壓瘡（褥瘡）往往是家屬最關心的議題之一。氣墊床（或稱交替式壓力氣墊床）正是為此而設計的重要輔具。所謂「氣墊床補助」，是指政府為減輕民眾購買或租賃氣墊床的經濟負擔，透過長期照顧（簡稱長照）服務體系，提供符合資格的民眾相關費用補助。這項補助旨在確保需要者能及時取得必要的輔具，提升照護品質，並有效預防壓瘡的發生。

## 為什麼重要

氣墊床對於長期臥床者而言，是不可或缺的照護輔具，其重要性體現在多個面向：

1.  **預防壓瘡（褥瘡）的關鍵：** 長期臥床者由於身體局部受壓時間過長，血液循環不良，極易形成壓瘡。氣墊床透過內建氣囊的交替充氣與洩氣，能不斷改變身體受壓部位，有效分散壓力，大幅降低壓瘡發生的風險。一旦壓瘡形成，不僅會造成患者極大痛苦，治療過程也漫長且耗費心力與金錢。
2.  **提升患者舒適度與生活品質：** 氣墊床的柔軟度與壓力分散功能，能讓臥床者感到更舒適，減少因長時間固定姿勢帶來的不適，進而提升其生活品質。
3.  **減輕照顧者負擔：** 氣墊床的輔助功能，能減少照顧者頻繁為患者翻身的需求，降低照顧者的身體勞動負擔，並讓他們有更多時間處理其他照護事宜。
4.  **經濟效益考量：** 一張高品質的氣墊床價格不菲，對於許多家庭而言是一筆沉重的開銷。政府提供的補助，能有效減輕家庭的經濟壓力，讓更多有需求的家庭能夠負擔得起這項重要的輔具，避免因經濟因素而延誤照護。

## 實際操作指南

如果您家中有臥床需求者，並考慮申請氣墊床補助，請依照以下步驟進行：

1.  **確認長照服務資格：**
    首先，您需要確認個案是否符合長照服務的資格。一般而言，只要是失能等級達到長照需要等級2級（含）以上，且符合相關條件（例如65歲以上老人、領有身心障礙證明者、55歲以上原住民、50歲以上失智症者等），即可申請長照服務。

2.  **聯絡當地長期照顧管理中心：**
    這是申請所有長照服務的第一步。您可以撥打全國統一專線1966，或直接聯繫您所在縣市的長期照顧管理中心。照管專員會初步了解您的需求，並安排到府評估。

3.  **照管專員到府評估：**
    照管專員會親自到府，針對個案的身體狀況、失能程度、居家環境以及輔具需求進行專業評估。在評估過程中，請務必明確告知照管專員個案有長期臥床、壓瘡風險或已發生壓瘡等情況，並表達對氣墊床的需求。照管專員會依據評估結果，將氣墊床納入個案的「照顧計畫」中。

4.  **核定與選購輔具：**
    一旦氣墊床需求被核定納入照顧計畫，照管中心會發給您「核定函」。請務必在收到核定函後，再前往「特約廠商」選購或租賃氣墊床。切記，**必須是與政府簽約的特約廠商**，才能申請補助。您可以向照管專員詢問特約廠商名單。

5.  **申請補助款核銷：**
    購買或租賃氣墊床後，請妥善保留發票或收據正本，並依據核定函上的指示，準備相關文件（例如身分證影本、存摺影本等），向照管中心提出補助款核銷申請。補助款通常會直接匯入申請人的指定帳戶。
